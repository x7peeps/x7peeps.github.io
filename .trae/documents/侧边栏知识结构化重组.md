## 二、AI — 整合旧内容到知识体系

### 问题

`chatgpt/` 和 `claude/` 两个目录是早期博客风格内容，与01-10知识体系割裂：
- 无 weight 字段
- tags 格式不一致（逗号分隔 vs YAML数组）
- 标题是操作指南风格

### 方案

将旧内容迁移到01-10体系中合适的子目录：

- `chatgpt/chatgpt微信机器人搭建/` → 移到 `09-实战项目架构/chatgpt微信机器人搭建/`
- `claude/claude注册/` → 移到 `01-LLM原理与工程/claude注册与配置/`
- 删除空的 `chatgpt/` 和 `claude/` 目录
- 更新迁移文章的 frontmatter（添加 weight、统一 tags 格式、修正 menu.parent）

## 三、执行步骤

1. 创建 0x03 下 4 个子分类目录及 `_index.md`
2. 用 `git mv` 将 88 篇平铺 .md 分批移动到对应子目录
3. 迁移 AI 的 chatgpt/claude 旧内容到 01-10 体系
4. 删除空目录
5. 更新迁移文章的 frontmatter
