# Forensics Articles Batch 36 - Spec

## 选题背景

蓝队取证分析深度文章系列已覆盖 49+ 个主题，从传统内存取证到前沿 AI/量子/卫星安全均有涉猎。本批次选择 3 个尚未覆盖的高价值前沿方向，聚焦新兴技术栈的安全取证分析。

## 选题理由

### 1. Rust语言安全取证深度分析 (weight: 1020)
- Rust 恶意软件在 2024-2026 年间快速增长，成为 APT 组织新偏好
- Rust 的所有权模型、零成本抽象、无 GC 特性给传统取证工具带来挑战
- Rust 二进制逆向分析、字符串提取、反混淆技术与 C/C++ 有本质差异
- 国内安全社区对 Rust 取证分析的系统性文章极为稀缺

### 2. WebAssembly安全取证深度分析 (weight: 1030)
- WASM 已从浏览器扩展到服务端、边缘计算、区块链智能合约
- WASM 二进制格式的混淆与反分析能力远超 JavaScript
- 浏览器端 WASM 挖矿、WASM 恶意载荷投递已成为常见攻击向量
- 缺乏系统性的 WASM 安全取证分析方法论

### 3. GPU与异构计算安全取证深度分析 (weight: 1040)
- GPU 显存取证是传统内存取证的盲区，需专用工具和方法
- CUDA 恶意计算、AI 模型窃取、GPU 侧信道攻击日益增多
- 加密货币挖矿木马广泛利用 GPU 异构计算能力
- NPU/TPU 等 AI 加速器的安全取证尚无系统性文章
