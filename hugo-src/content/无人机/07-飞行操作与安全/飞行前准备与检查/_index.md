# 飞行前准备与检查

飞行前准备与检查是保障无人机安全飞行的第一道防线，也是CAAC考试中关于飞行操作的核心考点之一。规范的航前检查流程不仅能有效预防飞行事故，更是专业无人机操控员的基本素养。本节将系统介绍飞行前准备的完整流程、检查要点以及常见的遗漏风险。

## 飞行前准备的完整流程

飞行前准备是一个系统化的过程，需要按照规范步骤逐步完成。完整的流程可分为以下四个阶段：

### 第一阶段：任务规划

在出发到飞行场地之前，需要完成以下准备工作：

1. **明确飞行任务目标**：确定本次飞行要完成的具体任务（航拍、测绘、巡检等）
2. **获取飞行许可**：根据飞行区域的空域要求，提前向相关部门申请飞行空域和时间
3. **制定飞行计划**：规划航线、确定飞行高度、预估飞行时间
4. **检查任务设备**：确认任务载荷（相机、传感器等）功能正常、存储空间充足

### 第二阶段：场地与天气评估

到达飞行场地后的第一项工作是评估飞行条件：

- 查看实时天气状况，确认风速、能见度等是否满足飞行要求
- 评估飞行场地的障碍物分布和人口密度
- 确认电磁环境，排除大功率无线电设备的干扰风险
- 检查起降场地的地面条件

### 第三阶段：设备检查

对无人机系统进行全面检查（详见下文航前检查清单）。

### 第四阶段：起飞前确认

完成所有检查后的最终确认：

- GPS搜星数量满足要求（通常要求至少10颗以上）
- 遥控器信号正常、电量充足
- 地面站/监控软件显示正常
- 确认返航点已正确设置
- 确认周围无安全隐患

## 飞行计划的制定要素

一份完整的飞行计划应包含以下要素：

| 要素 | 说明 | 重要性 |
|------|------|--------|
| 飞行目的 | 明确本次飞行的任务目标 | 决定飞行方式和设备选择 |
| 飞行区域 | 确定具体的飞行范围和边界 | 空域申请和安全控制的基础 |
| 飞行高度 | 设定计划飞行高度 | 须符合空域管理规定 |
| 飞行航线 | 规划具体的飞行路径 | 提高作业效率 |
| 预计时间 | 预估起飞时间和飞行时长 | 安排作业进度和电池管理 |
| 通信方案 | 确定与团队的通信方式 | 应急联络保障 |
| 应急预案 | 制定各类突发情况的处理方案 | 安全保障的关键 |
| 备降方案 | 设定紧急情况下的备降场地 | 应急处置的最后保障 |

## 航前检查清单

### 机体检查

| 检查项目 | 检查内容 | 合格标准 |
|---------|---------|---------|
| 机架结构 | 检查机身、机臂有无裂纹或变形 | 无可见损伤 |
| 螺丝紧固 | 检查所有结构螺丝是否紧固 | 无松动 |
| 桨叶检查 | 检查桨叶有无裂纹、缺口、变形 | 完好无损 |
| 电机检查 | 手动旋转电机，检查是否顺滑 | 无卡滞、异响 |
| 起落架 | 检查起落架结构完整性 | 牢固、无损伤 |
| 云台与相机 | 检查云台运动是否灵活 | 自由运动、无卡顿 |
| 标记与标识 | 确认无人机登记号、反光标识 | 清晰可见 |

### 电池检查

电池是无人机最重要的能源部件，电池相关的检查尤为重要：

1. **电量检查**：确认电池电量满足本次飞行任务需求（建议电量至少为预估飞行时间的1.5倍）
2. **外观检查**：检查电池外壳有无膨胀、变形、漏液
3. **接口检查**：检查电池接口有无氧化、腐蚀、松动
4. **温度检查**：确认电池温度在正常范围内（一般5-45°C）
5. **电芯一致性**：通过电池管理界面检查各电芯电压差（一般不超过0.1V）
6. **循环次数**：确认电池使用循环次数未超过厂家推荐的上限

{{< notice warning >}}
**考试易错点：** 低温环境下电池性能会显著下降。当环境温度低于10°C时，锂聚合物电池的实际可用容量可能降低20%-30%。在低温条件下飞行，需要适当减少飞行时间并做好电池保温措施。
{{< /notice >}}

### 遥控器检查

| 检查项目 | 检查内容 | 注意事项 |
|---------|---------|---------|
| 电量 | 确认遥控器电量充足 | 建议100%满电出发 |
| 天线 | 检查天线是否完好，安装是否正确 | 天线展开方向与信号传播方向垂直 |
| 指摇杆 | 测试各通道响应是否正常 | 各方向满行程无异常 |
| 开关设置 | 检查各功能开关位置 | 确认飞行模式开关在正确位置 |
| 返航设置 | 确认失控保护和返航设置 | 返航高度设置正确 |
| 固件版本 | 确认遥控器固件与飞行器匹配 | 版本一致 |

### 传感器校准

传感器校准是确保飞行控制系统正常工作的关键步骤：

- **指南针校准**：在远离金属和电磁干扰的开阔场地进行，每次更换飞行场地时建议重新校准
- **IMU校准**：将无人机放置在水平面上，按照提示完成加速度计和陀螺仪校准
- **气压计校准**：通常在起飞前自动完成，确认地面气压值读数正常

## 飞行场地评估

### 障碍物评估

| 障碍物类型 | 风险等级 | 应对措施 |
|-----------|---------|---------|
| 高压线 | 极高 | 避免在高压线附近飞行 |
| 树木 | 中等 | 保持足够的安全距离 |
| 建筑物 | 中等 | 评估GPS信号遮挡影响 |
| 通信塔 | 高 | 注意电磁干扰风险 |
| 鸟类活动区 | 中等 | 避开鸟类聚集区域和迁徙路线 |

### 电磁环境评估

无人机对电磁环境较为敏感，起飞前应确认：

1. 附近是否有大功率雷达站或通信基站
2. 是否有广播电视发射塔
3. 是否有高压输电线路
4. 是否有工业电磁干扰源
5. WiFi信号密度是否过高

### 人口密度评估

根据空域管理规定，在人口密集区域上方飞行需要满足更严格的安全要求：

- **A类人口密集区**：不得飞行或须获得特殊许可
- **B类区域**：须降低飞行高度并增加安全措施
- **C类区域**：可正常飞行，保持安全距离

## 天气条件判断

### 飞行安全气象标准

| 气象参数 | 安全飞行要求 | 警戒值 | 危险值 |
|---------|------------|--------|--------|
| 风速 | <6级（<10.8m/s） | 6-7级 | >7级 |
| 能见度 | >3km | 1-3km | <1km |
| 降水 | 无降水或小雨 | 中雨 | 大雨、冰雹 |
| 雷电 | 无雷电活动 | 远处有雷暴 | 近距离雷电 |
| 温度 | -10°C ~ 45°C | 边界值 | 超出范围 |

### 特殊天气注意事项

- **阵风**：比持续风更危险，可能导致无人机姿态突变
- **风切变**：低空风切变对无人机起降阶段威胁极大
- **雾霾**：影响能见度和操作员目视判断能力
- **高温**：可能导致电机过热、电池性能下降

## 飞行前GPS搜星要求

GPS搜星是飞行前准备的重要环节，直接关系到飞行安全：

### 搜星数量要求

| 搜星数量 | 定位状态 | 飞行建议 |
|---------|---------|---------|
| <6颗 | 无法定位 | 禁止起飞 |
| 6-9颗 | 基本定位 | 谨慎飞行，关注DOP值 |
| 10-15颗 | 良好定位 | 可正常飞行 |
| >15颗 | 优秀定位 | 最佳飞行条件 |

### DOP值确认

起飞前还应确认DOP值在合理范围内：

- **HDOP < 2.0**：水平定位精度优秀
- **VDOP < 3.0**：垂直定位精度良好
- **PDOP < 3.0**：综合定位精度良好

{{< notice info >}}
**实际操作要点：** GPS搜星需要一定时间（冷启动通常需要1-3分钟，热启动约10-30秒）。到达飞行场地后应第一时间开启无人机和遥控器进行搜星，在完成其他检查项目的同时等待搜星完成。切勿在搜星未完成的情况下匆忙起飞。
{{< /notice >}}

## 考试核心要点总结

| 考点 | 核心内容 |
|------|---------|
| 航前检查完整流程 | 任务规划→场地评估→设备检查→起飞前确认 |
| 电池检查要点 | 电量、外观、接口、温度、电芯一致性 |
| 指南针校准 | 远离金属和电磁干扰，每次更换场地建议重校 |
| GPS搜星要求 | 至少6颗以上，建议10颗以上，关注DOP值 |
| 天气判断标准 | 风速<6级、能见度>3km、无雷电 |
| 障碍物识别 | 高压线、通信塔为最高风险障碍物 |
| 搜星时间 | 冷启动1-3分钟，热启动10-30秒 |
