正确获取和解读航空气象信息是无人机安全飞行的关键环节。METAR、TAF等气象报文是航空领域获取实时和预报气象信息的标准方式，理解其格式和含义是CAAC考试的重要考点。掌握气象信息的获取渠道和应用方法，能够帮助飞手做出科学的飞行决策。

## 航空气象报文种类

### 主要报文类型

| 报文类型 | 全称 | 内容 | 有效期 | 更新频率 |
|----------|------|------|--------|---------|
| METAR | Meteorological Terminal Air Report | 机场例行天气实况 | 报发时至下一次METAR | 通常每小时 |
| SPECI | Special Aerodrome Weather Report | 机场特殊天气报告 | 实时 | 天气突变时发 |
| TAF | Terminal Aerodrome Forecast | 机场天气预报 | 12~30小时 | 每6~12小时 |
| SIGMET | Significant Meteorological Information | 重要气象信息 | 范围和时间不定 | 有重要天气时 |
| AIRMET | Airmen's Meteorological Information | 航空气象情报 | 区域性 | 有影响天气时 |

## METAR报文的格式和解读

### METAR报文结构

一份标准的METAR报文按以下顺序排列：

| 位置 | 项目 | 说明 | 示例 |
|------|------|------|------|
| 1 | 报文类型 | METAR或SPECI | METAR |
| 2 | 站点代码 | ICAO四字代码 | ZBAA |
| 3 | 观测时间 | 日期和时间（UTC） | 081200Z |
| 4 | 风向风速 | 风向（度）/风速（节） | 27008KT |
| 5 | 能见度 | 以米为单位 | 6000 |
| 6 | 天气现象 | 现在天气 | -RA BR |
| 7 | 云况 | 云量和云底高度 | BKN030 |
| 8 | 温度和露点 | 温度/露点（°C） | 18/12 |
| 9 | 气压 | 修正海平面气压 | QNH1013 |
| 10 | 趋势 | 未来2小时趋势 | BECMG TL1400 3000 |

### METAR报文详细示例

> **METAR ZBAA 081200Z 27008KT 6000 -RA BR BKN030 18/12 Q1013 BECMG TL1400 3000**

逐项解读：

| 报文片段 | 含义 |
|----------|------|
| METAR | 例行天气报告 |
| ZBAA | 北京首都国际机场 |
| 081200Z | 0日12时00分（UTC时间） |
| 27008KT | 风向270°（正西风），风速8节 |
| 6000 | 能见度6000米 |
| -RA | 小雨（-表示弱，RA表示雨） |
| BR | 轻雾/霾（能见度1~5km） |
| BKN030 | 多云，云底高度3000英尺（约914m） |
| 18/12 | 温度18°C，露点12°C |
| Q1013 | 修正海平面气压1013hPa |
| BECMG TL1400 3000 | 预计到14:00前转变为能见度3000m |

### 云量符号

| 符号 | 含义 | 云量比例 |
|------|------|---------|
| SKC / CLR | 晴空 | 0/8 |
| FEW | 少云 | 1/8~2/8 |
| SCT | 疏云 | 3/8~4/8 |
| BKN | 多云 | 5/8~7/8 |
| OVC | 阴天 | 8/8 |
| /// | 不可观测 | — |

### 云高表示

云高以百英尺为单位。例如：
- `FEW020` = 少云，云底高度2000英尺（约610m）
- `BKN030` = 多云，云底高度3000英尺（约914m）
- `OVC010` = 阴天，云底高度1000英尺（约305m）

### 常见天气现象符号

| 符号 | 含义 | 符号 | 含义 |
|------|------|------|------|
| +RA | 大雨 | -RA | 小雨 |
| RA | 雨 | TS | 雷暴 |
| SN | 雪 | -SN | 小雪 |
| FG | 雾 | BR | 轻雾 |
| HZ | 霾 | DS | 沙暴 |
| +TSRA | 雷暴伴大雨 | -SHRA | 阵雨 |

## TAF预报报文的格式和解读

### TAF报文结构

TAF报文提供机场未来一段时间的天气预报，格式与METAR类似但增加了时间范围指示。

### TAF报文示例

> **TAF ZBAA 080800Z 0809/0821 27010G20KT 6000 -RA BKN030 TEMPO 0812/0818 3000 RA BKN015 BECMG 0818/0821 29005KT 9999 SKC**

逐项解读：

| 报文片段 | 含义 |
|----------|------|
| TAF | 机场天气预报 |
| ZBAA | 北京首都国际机场 |
| 080800Z | 0日08时00分（UTC）发布 |
| 0809/0821 | 预报有效期：0日09时至21时 |
| 27010G20KT | 风向270°，持续风速10节，阵风20节 |
| 6000 | 能见度6000米 |
| -RA | 小雨 |
| BKN030 | 多云，云底3000英尺 |
| TEMPO 0812/0818 | 12时至18时期间可能暂时变化 |
| 3000 RA BKN015 | 能见度3000m，降雨，云底1500英尺 |
| BECMG 0818/0821 | 18时至21时期间逐渐转变 |
| 29005KT 9999 SKC | 风向290°风速5节，能见度10km以上，晴空 |

### TAF时间范围指示符

| 指示符 | 含义 | 说明 |
|--------|------|------|
| FM（From） | 从某时间起 | 持续至下一个变化指示 |
| TEMPO | 临时变化 | 变化持续时间不超过1小时 |
| BECMG | 逐渐转变 | 在指定时间段内逐渐变化 |
| INTER | 间歇性变化 | 短时间反复出现的变化 |

## 气象云图的基本判读

### 卫星云图

| 云图类型 | 波段 | 用途 |
|----------|------|------|
| 可见光云图 | 可见光波段 | 反映云的厚度和反射率 |
| 红外云图 | 红外波段 | 反映云顶温度和高度 |
| 水汽图 | 水汽吸收波段 | 反映中高层水汽分布 |

### 云图判读要点

- **白色越亮**：云层越厚、云顶越高
- **红外云图中白色**：云顶温度越低，对流发展越旺盛
- **带状云系**：通常与锋面系统相关
- **涡旋状云系**：可能对应气旋或台风

## 飞行天气标准

### VFR飞行条件

目视飞行规则（Visual Flight Rules, VFR）要求飞行中能够目视确认飞行状态和周围环境。

| 条件 | 要求 |
|------|------|
| 能见度 | 通常要求 ≥ 5km（视具体规定） |
| 云距 | 与云底保持安全距离 |
| 云高 | 不低于规定的最低云底高度 |

### IFR飞行条件

仪表飞行规则（Instrument Flight Rules, IFR）在能见度和云底高度不满足VFR条件时使用。

| 条件 | 说明 |
|------|------|
| 能见度 | 低于VFR标准 |
| 云底高度 | 低于VFR最低标准 |
| 导航方式 | 主要依靠仪表和导航设备 |

> **考试重点**：无人机目前主要执行目视飞行，因此了解VFR飞行条件非常重要。但随着无人机技术发展和BVLOS（超视距飞行）的推进，IFR飞行条件的理解也日益重要。

## 考试重点总结

### 核心考点

1. **METAR报文的正确解读**：按标准格式逐项解码各气象要素
2. **TAF报文的解读**：理解TEMPO、BECMG、FM等时间指示符的含义
3. **云量符号和云高表示**：SKC/FEW/SCT/BKN/OVC的区分
4. **VFR与IFR条件的区别**：了解各自的适用条件
5. **气象报文的更新频率和有效期**：METAR通常每小时更新，TAF每6~12小时

### 常见易错题型

| 题目类型 | 正确理解 | 常见错误 |
|----------|---------|---------|
| METAR中"27008KT"的含义 | 西风8节 | 东风8节 |
| BKN030的云底高度 | 3000英尺（约914m） | 300英尺 |
| TEMPO的含义 | 临时变化，不超过1小时 | 持续变化 |
| Q1013的含义 | 修正海平面气压1013hPa | 场面气压 |
| -RA的含义 | 小雨 | 大雨 |
| G在风速中的含义 | 阵风 | 持续风 |
