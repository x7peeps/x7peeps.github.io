危险天气是无人机飞行安全的最大威胁之一。雷暴、积冰、大雾和强风等恶劣天气条件可能导致无人机失控、坠毁甚至造成地面人员伤亡。识别和规避危险天气是每一位无人机操作员必须掌握的核心技能，也是CAAC考试中的高频考点。

## 雷暴

### 雷暴的形成条件

雷暴（Thunderstorm）的形成需要同时具备三个基本条件：

| 条件 | 说明 | 重要程度 |
|------|------|---------|
| 充足的水汽 | 低层大气含有大量水蒸气 | 必要条件 |
| 大气不稳定 | 温度随高度递减率大，有利于对流发展 | 必要条件 |
| 抬升机制 | 地形抬升、锋面抬升、热力对流等 | 触发条件 |

### 雷暴的发展阶段

| 阶段 | 持续时间 | 主要特征 | 对飞行的威胁 |
|------|---------|---------|-------------|
| 发展阶段（积云阶段） | 10~15分钟 | 上升气流为主，云体不断增长 | 中度湍流 |
| 成熟阶段 | 15~30分钟 | 上升和下沉气流并存，出现降水和闪电 | 最危险：雷击、冰雹、风切变、强湍流 |
| 消散阶段 | 30~60分钟 | 下沉气流为主，降水减弱 | 逐渐减弱，仍有下击暴流风险 |

### 雷暴的结构

一个典型雷暴的水平尺度为10~50km，垂直尺度可达10~15km（对流层内）。

| 部位 | 特征 | 飞行威胁 |
|------|------|---------|
| 积雨云主体 | 云顶可发展至平流层底部 | 严重湍流、积冰 |
| 云中闪电区 | 正负电荷分离产生放电 | 雷击损伤 |
| 冰雹区 | 云中部上升气流最强处 | 物理损伤 |
| 下击暴流 | 云底强烈下沉气流 | 极端风切变 |
| 飑线 | 雷暴前方强风带 | 突发强风 |

### 雷暴对无人机飞行的威胁

| 威胁类型 | 具体表现 | 危险程度 |
|----------|---------|---------|
| 雷击 | 直接击中或感应电流损伤电子设备 | 极高 |
| 风切变 | 雷暴内外的剧烈风场变化 | 极高 |
| 冰雹 | 高速冰块撞击机体 | 高 |
| 强湍流 | 上升和下沉气流交替造成剧烈颠簸 | 高 |
| 下击暴流 | 雷暴下方的强烈阵风 | 极高 |
| 大雨 | 严重影响能见度和设备工作 | 中 |

### 雷暴的躲避原则

1. **绕飞而非穿越**：永远不要试图穿越雷暴，应绕飞至少20km以上
2. **远离雷暴中心**：保持与雷暴单体的安全距离
3. **关注飑线**：飑线前方的强风可能在雷暴主体到达之前就产生影响
4. **不要降落在雷暴区下方**：下击暴流可能在降水区前方出现
5. **关注雷达回波**：利用气象雷达判断雷暴的强度和移动方向

> **考试重点**：绝对不允许无人机穿越雷暴。即使是弱雷暴也可能产生致命的风切变和雷击。躲避距离应至少保持20km。

## 飞机积冰

### 积冰的条件

飞机积冰需要同时满足以下条件：

| 条件 | 说明 |
|------|------|
| 含有过冷水滴的云或降水 | 温度在0°C以下但仍保持液态的水滴 |
| 机体表面温度低于0°C | 过冷水滴碰到机体后冻结 |
| 一定的飞行时间 | 积冰是一个逐渐累积的过程 |

### 积冰的类型

| 积冰类型 | 形成条件 | 外观特征 | 危险程度 |
|----------|---------|---------|---------|
| 明冰（Clear Ice） | 大过冷水滴在温度略低于0°C时冻结 | 透明或半透明，光滑坚硬 | 极高，最难清除 |
| 雾凇冰（Rime Ice） | 小过冷水滴在温度远低于0°C时冻结 | 不透明，表面粗糙，白色 | 较高，影响气动外形 |
| 混合冰（Mixed Ice） | 明冰和雾凇冰的混合 | 介于两者之间 | 高 |

### 各类积冰的详细对比

| 对比项 | 明冰 | 雾凇冰 | 混合冰 |
|--------|------|--------|--------|
| 水滴大小 | 大水滴 | 小水滴 | 混合 |
| 温度范围 | 0°C ~ -10°C | -10°C ~ -20°C | 两者之间 |
| 冻结速度 | 缓慢，水滴摊开后冻结 | 快速，碰到即冻结 | 介于两者 |
| 附着力 | 强，不易脱落 | 较弱 | 中等 |
| 对气动的影响 | 严重改变翼型 | 轻度改变翼型 | 中等 |

### 积冰对飞行性能的影响

| 影响方面 | 具体表现 |
|----------|---------|
| 升力减小 | 冰层改变了机翼/旋翼的翼型，降低升力系数 |
| 阻力增大 | 冰层增加表面粗糙度和截面积 |
| 重量增加 | 冰层累积增加机体重量 |
| 动力损失 | 螺旋桨积冰影响推进效率 |
| 传感器失效 | 天线、皮托管等被冰堵塞 |
| 操纵受限 | 操纵面积冰可能卡死或效率下降 |

> **考试易错点**：积冰最容易发生在温度0°C至-20°C的范围内。温度过低（低于-20°C）时，空气中含有的过冷水滴较少，积冰反而不严重。0°C附近的大水滴是最危险的积冰条件。

## 雾

### 雾的分类

雾是近地面层空气中水汽凝结形成的悬浮水滴或冰晶，使能见度降至1km以下。

| 雾的类型 | 形成原因 | 特征 |
|----------|---------|------|
| 辐射雾 | 夜间地面辐射冷却 | 晴朗微风的夜间形成，日出后逐渐消散 |
| 平流雾 | 暖湿空气流经冷地面 | 范围广，持续时间长 |
| 蒸发雾 | 冷空气流经暖水面 | 秋冬季节水面附近 |
| 上坡雾 | 潮湿空气沿地形爬升冷却 | 山区常见 |
| 锋面雾 | 锋面附近暖湿空气冷却 | 伴随锋面降水 |

### 雾对飞行的影响

| 影响 | 说明 |
|------|------|
| 能见度急剧降低 | 可能降至几百米甚至几十米 |
| 起飞着陆困难 | 视觉参考不足 |
| 定位困难 | 无法目视确认位置 |
| 积冰风险 | 雾滴温度低于0°C时可能积冰 |

## 低云低能见度天气

### 低云低能见度的成因

低云和低能见度通常由以下天气系统导致：

- **暖锋降水区**：大面积的层状云和持续性降水
- **雾区**：近地面水汽凝结
- **霾**：悬浮颗粒物降低能见度
- **沙尘**：强风扬起沙尘

### 对飞行的影响

低云低能见度天气可能导致：

- 无法满足目视飞行条件
- 定位和导航困难
- 无法确认飞行状态
- 无法目视判断与其他物体的安全距离

## 强风天气标准

### 各类风的标准

| 风况 | 风速条件 | 对无人机的影响 |
|------|---------|---------------|
| 5级风（清劲风） | 8.0~10.7 m/s | 小型无人机应停止飞行 |
| 6级风（强风） | 10.8~13.8 m/s | 中型无人机受严重影响 |
| 7级风（疾风） | 13.9~17.1 m/s | 大多数无人机无法安全飞行 |
| 8级风（大风） | 17.2~20.7 m/s | 禁止所有无人机飞行 |

### 阵风的影响

即使平均风在安全范围内，强阵风仍可能超出无人机的抗风能力。评估风况时应以阵风风速为准。

> **考试注意**：无人机的抗风能力通常以"持续风速"标称，但实际飞行中还需考虑阵风。例如标称抗风5级的无人机，在4级平均风但阵风6级的条件下可能无法安全飞行。

## 考试重点总结

### 核心考点

1. **积冰条件判断**：需要过冷水滴、低温表面和足够时间三个条件同时满足
2. **雷暴躲避策略**：绕飞而非穿越，保持安全距离至少20km
3. **积冰类型区分**：明冰、雾凇冰、混合冰的形成条件和特征差异
4. **雾的分类和影响**：各类雾的形成条件及对能见度的影响
5. **强风天气标准**：不同风速等级对无人机飞行的限制

### 易混淆辨析

| 概念A | 概念B | 关键区别 |
|-------|-------|---------|
| 明冰 | 雾凇冰 | 大水滴低温冻结 vs 小水滴快速冻结 |
| 辐射雾 | 平流雾 | 冷却形成 vs 暖湿平流形成 |
| 发展阶段 | 成熟阶段 | 上升气流为主 vs 上升下沉并存 |
| 积冰最危险温度 | 积冰最少温度 | 0°C~-20°C最危险 vs <-40°C较少 |
| 雷暴绕飞距离 | 安全飞行距离 | 至少20km vs 取决于雷暴大小 |
