## 风的形成原因和分类

### 风的形成

风是空气从高气压区域向低气压区域水平流动的现象。其形成的根本原因是太阳对地球表面的不均匀加热，导致不同区域气压差异，空气从高压区流向低压区。

### 按时间和空间尺度分类

| 风的类型 | 产生原因 | 尺度 | 持续时间 |
|----------|---------|------|---------|
| 大气环流 | 太阳辐射不均 | 全球尺度 | 长期 |
| 季风 | 海陆热力差异 | 大陆尺度 | 季节性 |
| 海陆风 | 海陆昼夜温差 | 地方性 | 日变化 |
| 山谷风 | 山坡与谷地温差 | 地方性 | 日变化 |
| 阵风 | 对流或障碍物 | 局部 | 短暂 |

### 按高度分层

| 风层 | 高度范围 | 特点 |
|------|---------|------|
| 地面风 | 地面至10m | 受地形和摩擦影响最大 |
| 近地面风 | 10~100m | 摩擦影响逐渐减小 |
| 梯度风 | 约1000m以上 | 不受地面摩擦影响 |
| 高空风 | 对流层中上部 | 地转风或梯度风 |

## 风的表示方法

### 风向

风向是指风吹来的方向，以正北为基准顺时针度量，用360°或16个方位表示。

| 表示方式 | 示例 | 含义 |
|----------|------|------|
| 角度表示 | 270° | 风从正西方向吹来 |
| 方位表示 | W（西风） | 风从正西方向吹来 |
| 简写表示 | 270/15 | 风向270°，风速15节 |

### 风速

风速是单位时间内空气移动的距离，常用单位有m/s、km/h和节（knot）。

| 单位换算 | 关系 |
|----------|------|
| 1节 = 1海里/小时 | ≈ 0.514 m/s |
| 1 m/s | ≈ 1.944 节 ≈ 3.6 km/h |
| 1 km/h | ≈ 0.278 m/s ≈ 0.540 节 |

### 蒲福风力等级

| 风力等级 | 风速范围（m/s） | 风速范围（节） | 描述 |
|----------|----------------|----------------|------|
| 0 | 0~0.2 | 0~0.5 | 无风 |
| 1 | 0.3~1.5 | 1~3 | 软风 |
| 2 | 1.6~3.3 | 4~6 | 轻风 |
| 3 | 3.4~5.4 | 7~10 | 微风 |
| 4 | 5.5~7.9 | 11~16 | 和风 |
| 5 | 8.0~10.7 | 17~21 | 清劲风 |
| 6 | 10.8~13.8 | 22~27 | 强风 |

## 地转风与梯度风

### 地转风

地转风是气压梯度力与科氏力（地转偏向力）平衡时产生的水平气流。地转风的特征是风向平行于等压线，低压在风向左侧（北半球）。

| 特征 | 说明 |
|------|------|
| 形成条件 | 气压梯度力 = 科氏力 |
| 风向特点 | 平行于等压线 |
| 高度条件 | 约1000m以上，不受地面摩擦影响 |
| 应用 | 高空天气分析 |

### 梯度风

梯度风是在地转风基础上考虑了等压线弯曲效应的风。在气旋（低压中心）和反气旋（高压中心）附近，等压线弯曲使气流需要额外的向心力。

## 风切变

### 风切变的定义

风切变（Wind Shear）是指在短距离内风向或风速发生突然变化的现象。风切变可以发生在水平方向或垂直方向，对飞行安全构成严重威胁。

### 风切变的分类

#### 按空间方向分类

| 类型 | 定义 | 对飞行的影响 |
|------|------|-------------|
| 水平风切变 | 同一高度水平方向上风速或风向的突然变化 | 导致空速突然变化 |
| 垂直风切变 | 垂直方向上风速或风向的突然变化 | 导致升力突然变化 |

#### 按气象成因分类

| 类型 | 形成原因 | 危险程度 |
|------|---------|---------|
| 雷暴下击暴流 | 积雨云中强烈的下沉气流 | 极其危险 |
| 锋面风切变 | 冷暖气团交界处 | 高度危险 |
| 低空急流风切变 | 低空急流的风速变化 | 较危险 |
| 地形风切变 | 地形对气流的改变 | 中度危险 |
| 辐射逆温风切变 | 夜间近地面逆温层 | 低空常见 |

### 风切变的危害

风切变对无人机飞行的危害主要体现在以下几个方面：

| 危害阶段 | 具体表现 | 后果 |
|----------|---------|------|
| 进入顺风切变 | 空速突然减小 | 升力骤降，高度下降 |
| 进入逆风切变 | 空速突然增大 | 升力骤增，高度上升 |
| 垂直风切变 | 上升或下沉气流突变 | 无法控制的上升或下沉 |

#### 顺风切变的危险过程

1. 无人机从逆风区域进入顺风区域
2. 空速突然减小（等于原空速减去风速变化量）
3. 升力与空速的平方成正比，空速减小导致升力大幅下降
4. 无人机快速下沉
5. 如果高度不足，可能导致触地坠毁

> **考试易错点**：风切变最危险的场景是在低空（起飞和着陆阶段），因为此时无人机高度低、速度低，没有足够的空间和时间来恢复。但在高空同样需要警惕风切变的影响。

### 低空风切变的识别

| 识别线索 | 说明 |
|----------|------|
| 阵风风向突变 | 地面风向短时间内变化超过45° |
| 温度骤变 | 气温突然升高或降低可能预示锋面接近 |
| 降水变化 | 降水强度突变伴随风场变化 |
| 云底变化 | 云底高度快速变化 |
| 气压变化 | 气压趋势异常 |

### 风切变的应对原则

1. **提前获取气象信息**：飞行前充分了解天气条件
2. **保持高度警惕**：在可能出现风切变的区域保持警觉
3. **保持充足动力余量**：确保有足够的功率储备应对突变
4. **发现风切变立即应对**：增大推力、调整姿态、尽快脱离

## 湍流

### 湍流的类型

| 湍流类型 | 产生原因 | 典型场景 |
|----------|---------|---------|
| 热力湍流 | 地面不均匀加热导致的空气对流 | 晴天午后陆地上空 |
| 动力湍流 | 气流经过障碍物（地形、建筑物）后的扰动 | 山区、城市建筑群 |
| 晴空湍流 | 高空急流附近的风切变 | 高空无云区域 |
| 尾流湍流 | 其他飞行器尾部产生的旋转气流 | 机场附近、航路 |
| 对流湍流 | 积云和积雨云内的强烈上升/下沉气流 | 对流旺盛天气 |

### 湍流对无人机的影响

- **姿态不稳定**：颠簸导致姿态控制系统频繁调整
- **结构应力**：突然的载荷变化增加结构疲劳
- **设备影响**：振动可能影响相机稳定性和传感器精度
- **通信干扰**：强烈湍流可能伴随电磁干扰

## 阵风与平均风

### 基本概念

| 概念 | 定义 | 说明 |
|------|------|------|
| 平均风 | 一段时间内风速的平均值 | 反映风的总体趋势 |
| 阵风 | 短时间内风速的峰值 | 通常是平均风的1.5~2倍 |
| 阵风因子 | 阵风风速 / 平均风速 | 通常为1.5~2.5 |

### 阵风对飞行的影响

- **突然的空速变化**：阵风导致空速波动，影响升力稳定性
- **姿态扰动**：阵风产生的力矩导致姿态突然偏转
- **航迹偏差**：阵风使无人机偏离预定航线
- **功率需求波动**：电机需要频繁调整转速应对阵风

> **考试重点**：飞行中应以阵风风速而非平均风速来评估飞行安全条件。例如平均风速3级（5m/s），但阵风可能达到5级（10m/s），应按5级风的条件来评估安全裕度。

## 风对无人机飞行的具体影响

### 对多旋翼无人机

| 风况 | 影响 | 应对措施 |
|------|------|---------|
| 逆风 | 地速减小，续航缩短 | 增加电池容量或缩短任务范围 |
| 顺风 | 地速增大，续航增加 | 注意返航时变为逆风 |
| 侧风 | 航线偏移，持续偏转消耗功率 | 自动偏航修正，增加电量裕度 |
| 阵风 | 姿态不稳，空速波动 | 增加悬停高度裕度 |

### 对固定翼无人机

| 风况 | 影响 | 应对措施 |
|------|------|---------|
| 逆风 | 地速大幅减小 | 提高空速或缩短任务 |
| 顺风 | 地速大幅增加 | 降落时减小地速 |
| 侧风 | 偏流角修正 | 侧风修正技术 |
| 阵风 | 升力波动，高度变化 | 保持足够高度裕度 |

## 考试重点总结

### 核心考点

1. **风切变的危害和识别**：特别是低空风切变对起飞和着陆的威胁
2. **风向和风速的表示方法**：角度制、方位制、蒲福风力等级
3. **阵风与平均风的区别**：评估飞行安全应以阵风为基准
4. **湍流的分类**：热力湍流、动力湍流、晴空湍流等的成因
5. **风对不同机型的具体影响**：多旋翼和固定翼的差异

### 易混淆概念

| 概念A | 概念B | 关键区别 |
|-------|-------|---------|
| 地转风 | 梯度风 | 直线等压线 vs 弯曲等压线 |
| 水平风切变 | 垂直风切变 | 同层风变 vs 垂直方向风变 |
| 平均风 | 阵风 | 长期平均 vs 瞬时峰值 |
| 热力湍流 | 动力湍流 | 对流产生 vs 障碍物产生 |
| 逆风切变 | 顺风切变 | 空速增大 vs 空速减小 |
