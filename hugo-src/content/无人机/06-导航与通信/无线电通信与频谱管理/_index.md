# 无线电通信与频谱管理

无线电通信是无人机系统中不可或缺的组成部分，涵盖了遥控链路、数据链路、图传链路等多个方面。了解不同频段的特性、频谱管理规则以及干扰应对措施，是保障无人机安全飞行和通过CAAC考试的基础知识。

## 无人机通信使用的频段

无人机系统中涉及多种无线电通信频段，根据用途可以分为以下几类：

| 通信类型 | 常用频段 | 典型用途 |
|---------|---------|---------|
| 遥控链路 | 2.4GHz、5.8GHz | 地面站向无人机发送控制指令 |
| 数据链路 | 900MHz、1.4GHz、2.4GHz | 双向传输遥测数据和任务数据 |
| 图传链路 | 2.4GHz、5.8GHz | 无人机向地面传输实时视频 |
| 卫星通信 | L波段、S波段、Ku波段 | 远程无人机的超视距通信 |
| 避障传感器 | 24GHz、60GHz、激光 | 近距离障碍物检测 |

### 频率与波长的关系

无线电波的频率和波长成反比关系：

- 波长 = 光速 / 频率
- 频率越高，波长越短，天线尺寸越小
- 频率越低，波长越长，传播距离越远

## ISM频段详解

### 2.4GHz频段

**ISM**（Industrial, Scientific and Medical）频段即工业、科学和医疗频段，是全球通用的免许可频段。2.4GHz是无人机最常用的频段。

2.4GHz频段的特性：

| 特性 | 说明 |
|------|------|
| 频率范围 | 2.4000-2.4835 GHz |
| 带宽 | 约83.5 MHz |
| 信道数量 | 通常分为13-14个信道（每个20/22 MHz） |
| 穿透能力 | 中等，可穿透部分墙壁和障碍物 |
| 传输距离 | 较5.8GHz更远，典型视距范围1-10km |
| 干扰程度 | 严重（WiFi、蓝牙、微波炉等均在此频段） |

### 5.8GHz频段

5.8GHz频段是另一个常用的无人机通信频段。

5.8GHz频段的特性：

| 特性 | 说明 |
|------|------|
| 频率范围 | 5.725-5.875 GHz |
| 带宽 | 约150 MHz |
| 信道数量 | 更多可用信道 |
| 穿透能力 | 较弱，障碍物衰减大 |
| 传输距离 | 较2.4GHz近，但干扰较少 |
| 干扰程度 | 轻（WiFi 5GHz设备逐渐增多） |

### 2.4GHz与5.8GHz对比

| 对比项 | 2.4GHz | 5.8GHz |
|--------|--------|--------|
| 传输距离 | 远 | 近 |
| 穿透能力 | 强 | 弱 |
| 可用带宽 | 窄 | 宽 |
| 干扰程度 | 严重 | 较轻 |
| 图传画质 | 较低 | 较高 |
| 适用场景 | 远距离飞行 | 近距离高画质图传 |

## 不同频段的传播特性对比

### 自由空间路径损耗

无线电波在传播过程中，信号强度会随距离增大而衰减，这就是**自由空间路径损耗**（FSPL）。

频率越高，自由空间路径损耗越大——这就是为什么5.8GHz的传输距离不如2.4GHz的根本原因。

主要频段传播特性对比：

| 频段 | 优点 | 缺点 | 典型应用 |
|------|------|------|---------|
| 433MHz | 传输距离远、穿透力强 | 带宽极低、天线大 | 遥测数据链路 |
| 900MHz | 距离与带宽平衡好 | 频段资源有限 | 长距离数据链路 |
| 1.4GHz | 距离适中、带宽适中 | 使用较少 | 专用数据链路 |
| 2.4GHz | 距离远、天线小 | 干扰严重 | 遥控、图传 |
| 5.8GHz | 干扰少、带宽大 | 距离近、穿透差 | 高清图传 |
| 毫米波 | 极高带宽 | 距离极短、衰减极大 | 毫米波雷达避障 |

### 影响传播距离的因素

1. **发射功率**：功率越大，传输距离越远
2. **天线增益**：高增益天线可以集中信号能量
3. **工作频率**：频率越高，路径损耗越大
4. **环境遮挡**：建筑物、植被等会吸收和反射信号
5. **大气条件**：雨天、雾天对高频段信号影响显著

## 频谱管理的基本原则

### 频谱管理的重要性

无线电频谱是有限的国家资源，任何无线电设备的使用都必须遵守频谱管理规定。

### 无人机频谱管理的基本原则

1. **合法使用**：使用经国家无线电管理机构批准的频段和发射参数
2. **不干扰**：不得对其他合法无线电业务造成有害干扰
3. **不受干扰**：有权免受其他设备的有害干扰
4. **执照要求**：特定频段的使用需要申请无线电频率使用许可
5. **技术参数限制**：发射功率、占用带宽、杂散发射等须符合国家标准

### 中国无人机常用频段的管理要求

| 频段 | 管理要求 |
|------|---------|
| 433MHz | 需申请频率使用许可 |
| 2.4GHz ISM | 免许可，但须遵守功率限制 |
| 5.8GHz ISM | 免许可，但须遵守功率限制 |
| 840-845MHz | 遥控航空模型专用频段 |

{{< notice warning >}}
**考试重点：** 2.4GHz和5.8GHz是ISM免许可频段，但使用时仍须遵守功率限制和杂散发射标准。考试中经常考察哪些频段需要许可、哪些频段免许可的区别。
{{< /notice >}}

## 无线电干扰的类型和应对

### 干扰类型

| 干扰类型 | 说明 | 常见来源 |
|---------|------|---------|
| 同频干扰 | 其他设备在同一频率上发射信号 | 同一区域的其他无人机、WiFi路由器 |
| 邻频干扰 | 相邻频率的信号泄漏到工作频段 | 大功率基站、其他无线电设备 |
| 阻塞干扰 | 强信号导致接收机前端饱和 | 大功率雷达、广播电视发射机 |
| 谐波干扰 | 非线性器件产生的谐波落入工作频段 | 劣质电子设备 |
| 互调干扰 | 多个信号在非线性器件中产生互调产物 | 多台设备同时工作 |

### 干扰应对措施

1. **跳频技术**（FHSS）：在多个频率之间快速切换，避开被干扰的频率
2. **直接序列扩频**（DSSS）：将信号扩展到更宽的频带上，提高抗干扰能力
3. **自适应选频**：实时检测频谱环境，自动选择干扰最小的频段工作
4. **滤波器**：使用带通滤波器滤除工作频段外的干扰信号
5. **天线方向性**：使用定向天线减少来自其他方向的干扰

## 无人机反制技术简述

无人机反制技术是针对非法或威胁性无人机的对抗手段。了解这些技术有助于无人机操作员理解安全风险并做好防范。

### 主要反制手段

| 反制方式 | 原理 | 效果 |
|---------|------|------|
| 无线电干扰 | 发射大功率干扰信号，阻断遥控或导航链路 | 迫使无人机返航或悬停 |
| GPS欺骗 | 发射虚假GPS信号，诱导无人机飞离目标区域 | 改变无人机飞行轨迹 |
| 协议破解 | 截获并破解无人机通信协议，获取控制权 | 劫持无人机 |
| 激光武器 | 高能激光烧毁无人机结构或传感器 | 物理摧毁 |
| 网捕获 | 使用捕获网拦截无人机 | 物理捕获 |

{{< notice info >}}
**安全意识：** 了解反制技术不仅有助于考试，更有助于操作员在实际飞行中识别和应对潜在的安全威胁。例如，GPS欺骗可能导致无人机位置漂移，操作员应具备识别异常位置偏差的能力。
{{< /notice >}}

## 数据链路的带宽和传输距离关系

### 带宽-距离的权衡

在无线通信中，带宽和传输距离存在基本的权衡关系：

- **高带宽**：需要更宽的信号频带和更高的信噪比，传输距离较短
- **远距离**：需要窄带宽和高接收灵敏度来维持链路

### 无人机数据链路的实际考虑

| 应用场景 | 带宽需求 | 推荐频段 | 预期距离 |
|---------|---------|---------|---------|
| 低速遥测 | <100 kbps | 433/900 MHz | 5-20 km |
| 标清图传 | 1-4 Mbps | 2.4 GHz | 1-5 km |
| 高清图传 | 10-50 Mbps | 5.8 GHz | 0.5-2 km |
| 4K图传 | 50-100 Mbps | 5.8 GHz | 0.3-1 km |
| 超视距通信 | 1-10 Mbps | 卫星频段 | 100+ km |

### 链路预算

链路预算是评估通信系统能否建立可靠连接的关键工具：

**链路预算基本公式：**

- 接收功率 = 发射功率 + 发射天线增益 + 接收天线增益 - 路径损耗 - 其他损耗

接收功率必须大于接收机灵敏度才能保证正常通信，二者之间的差值即为**链路余量**。

链路余量越大，通信越可靠。通常需要保留10-20 dB的链路余量以应对环境变化。

## 考试核心要点总结

| 考点 | 核心内容 |
|------|---------|
| 频段选择 | 2.4GHz传得远但干扰多，5.8GHz干扰少但距离近 |
| ISM频段 | 2.4GHz和5.8GHz为免许可频段，但仍须遵守功率限制 |
| 干扰类型 | 同频干扰、邻频干扰、阻塞干扰等 |
| 跳频技术 | 通过快速跳变频率来避开干扰频段 |
| 反制技术 | 无线电干扰、GPS欺骗、协议破解等 |
| 带宽-距离关系 | 带宽越高，传输距离越近 |
| 链路预算 | 接收功率须大于接收机灵敏度，需留有余量 |
