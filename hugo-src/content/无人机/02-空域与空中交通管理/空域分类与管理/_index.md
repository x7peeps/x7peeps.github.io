# 空域分类与管理

空域是国家的重要自然资源，是无人机飞行活动的基础载体。理解空域的分类体系、管理体制以及相关飞行规则，是 CAAC 无人机执照考试的核心考点之一。本节将系统梳理我国空域分类的完整框架，重点解析考试中常见的易混淆概念和高频考点。

---

## 1. 空域的基本概念

### 1.1 什么是空域

空域是指地球表面以上可供航空器飞行的空中空间。空域并非无限开放的公共空间，而是由国家统一管理和划分的有限资源。任何航空器——包括有人驾驶飞机和无人机——都必须在被允许的空域范围内飞行。

### 1.2 空域的两大分类

根据是否实施空中交通管制，空域分为两大类：

| 分类 | 定义 | 特征 |
| :--- | :--- | :--- |
| **管制空域** | 实施空中交通管制服务的空域 | 飞行需获得管制许可，有明确的飞行规则和间隔标准 |
| **非管制空域** | 不实施空中交通管制服务的空域 | 飞行器自行负责安全，但仍需遵守基本飞行规则 |

**考试易错点**：非管制空域并不意味着"随意飞行"，飞行器驾驶员仍然需要遵守航行情报服务的相关规定，并保持适当的空中观察。

## 2. 我国空域管理体制

### 2.1 管理原则

我国空域管理实行**军民分管**的体制，核心原则为：

- **中国人民解放军空军**统一领导全国的飞行管制
- **中国民用航空局（CAAC）**负责民用航空活动的空域使用管理
- 空域的划设和调整由空军会同民航部门共同确定

### 2.2 管理架构

| 层级 | 负责单位 | 职责 |
| :--- | :--- | :--- |
| **国家级** | 空军、民航局 | 制定空域管理政策和标准 |
| **区域级** | 军区空军、地区管理局 | 区域空域的划设和日常管理 |
| **机场级** | 机场塔台、军航调度 | 具体空域的使用协调和管制 |

**考试重点**：空域管理的最高领导权属于**空军**，而非民航局。民航局是在空军统一领导下的民用航空空域管理部门。

## 3. 我国空域分类详解

### 3.1 空域分类体系

根据《中华人民共和国飞行基本规则》和相关法规，我国空域划分为以下类型：

| 空域类型 | 代号 | 适用范围 | 管制要求 |
| :--- | :--- | :--- | :--- |
| **管制空域** | A类 | 高空飞行空域 | 实施雷达管制，提供全套管制服务 |
| **管制空域** | B类 | 中低空飞行空域 | 实施雷达或程序管制 |
| **管制空域** | C类 | 进近管制空域 | 机场附近区域，实施进近管制 |
| **管制空域** | D类 | 机场管制空域 | 以机场为中心，塔台管制 |
| **非管制空域** | E类 | 一般飞行空域 | 不实施管制，提供飞行情报 |
| **非管制空域** | G类 | 低空飞行空域 | 最基本的飞行规则 |

### 3.2 各类空域的特点对比

| 特征 | A类 | B类 | C类 | D类 | E类 | G类 |
| :--- | :--- | :--- | :--- | :--- | :--- | :--- |
| **高度范围** | 6000m以上 | 6000m以下 | 机场终端区 | 机场周边 | 未划入其他类 | 低空区域 |
| **管制级别** | 最高 | 高 | 较高 | 高 | 无 | 无 |
| **通信要求** | 强制双向 | 双向 | 双向 | 双向 | 建议 | 不强制 |
| **气象要求** | 严格 | 较严格 | 严格 | 严格 | 一般 | 一般 |
| **间隔标准** | 雷达间隔 | 雷达/程序 | 程序/目视 | 目视/程序 | 无强制 | 无强制 |

### 3.3 无人机可飞行空域

对于无人机运营，可飞行空域有特殊规定：

- **微型无人机**：可在非管制空域飞行，飞行高度不超过 120 米
- **轻型无人机**：需在指定空域飞行，需向相关部门报备
- **中型及以上无人机**：必须在管制空域内飞行，需获得管制许可

**考试高频考点**：无人机飞行的"120 米"高度限制是指**真高度**（AGL，Above Ground Level），即距离正下方地面的垂直高度，而非海拔高度。

## 4. 真高度与相对高度——考试易混淆点

这是 CAAC 考试中最容易混淆的概念之一，务必牢记：

### 4.1 概念区分

| 概念 | 英文 | 定义 | 计算基准 |
| :--- | :--- | :--- | :--- |
| **真高度（AGL）** | Above Ground Level | 航空器距离**正下方地面**的垂直高度 | 地面 |
| **相对高度（MSL）** | Mean Sea Level | 航空器距离**平均海平面**的垂直高度 | 海平面 |
| **标准气压高度** | Pressure Altitude | 以标准气压面（1013.25 hPa）为基准的高度 | 标准气压面 |
| **场面气压高度** | QFE Altitude | 以机场场面气压为基准的高度 | 机场场面 |

### 4.2 举例说明

假设某机场海拔高度为 500 米，一架无人机在机场上方 300 米处飞行：

- 真高度（AGL）= 300 米
- 相对高度（MSL）= 500 + 300 = **800 米**

**考试易错点**：在山地区域飞行时，真高度和相对高度的差值会显著增大。无人机限高 120 米指的是真高度，即使你在高海拔地区起飞，120 米的限制仍然是相对于起飞点正下方地面的高度。

## 5. 高度层配备标准

### 5.1 RVSM 区域

**RVSM（Reduced Vertical Separation Minimum）**，即缩小垂直间隔标准：

- 适用高度范围：FL290（8850 米）至 FL410（12500 米）
- 垂直间隔：由原来的 600 米（2000 英尺）缩小为 **300 米（1000 英尺）**
- 适用条件：航空器需具备 RVSM 适航认证

### 5.2 非 RVSM 区域

- FL290 以下：垂直间隔为 **300 米（1000 英尺）**
- FL410 以上：垂直间隔为 **600 米（2000 英尺）**

### 5.3 高度层配备规则

| 飞行方向 | 真航向范围 | 高度层配备 |
| :--- | :--- | :--- |
| **东行** | 000° - 179° | 奇数高度层（如 900m、1500m、2100m...） |
| **西行** | 180° - 359° | 偶数高度层（如 600m、1200m、1800m...） |

**记忆口诀**："东奇西偶"——向东飞用奇数高度层，向西飞用偶数高度层。

**考试注意**：此规则适用于我国境内飞行的航空器，且仅适用于仪表飞行规则（IFR）条件下的飞行。目视飞行规则（VFR）有独立的高度层使用规定。

## 6. 考试重点总结

### 6.1 空域分类判断方法

1. **首先判断空域类别**：管制空域（A/B/C/D）还是非管制空域（E/G）
2. **确认飞行高度**：是否在限制高度范围内
3. **检查许可要求**：是否需要管制许可
4. **核实通信设备**：是否具备所需通信能力

### 6.2 常见考点归纳

| 考点 | 关键要点 |
| :--- | :--- |
| 空域管理最高机构 | 中国人民解放军空军 |
| 无人机限高 120 米 | 指真高度（AGL），非海拔高度 |
| 高度层"东奇西偶" | 真航向 000°-179° 用奇数，180°-359° 用偶数 |
| RVSM 缩小间隔高度 | FL290 至 FL410，间隔 300 米 |
| 非管制空域 | 仍需遵守飞行规则，不是"自由飞行" |
| 管制空域分类 | A（高空）/ B（中低空）/ C（进近）/ D（机场） |
| 微型无人机飞行 | 非管制空域，真高度不超过 120 米 |

### 6.3 易混淆概念辨析

- **真高度 vs 海拔高度**：真高度以地面为基准，海拔以海平面为基准
- **管制空域 vs 控制区**：管制空域是管理分类，控制区是空域结构概念
- **飞行许可 vs 空域使用许可**：前者是管制单位签发，后者是空域管理部门签发
- **RVSM 间隔 vs 标准间隔**：RVSM 区域内间隔为 300 米，其他区域通常为 600 米或 300 米

掌握以上概念和判断方法，能够帮助考生在面对空域分类相关题目时快速做出正确判断。建议结合实际空域图进行练习，加深对各类空域划分的直观理解。
