# 飞行情报与航行通告

飞行情报服务是保障飞行安全的重要环节，而航行通告（NOTAM）则是飞行员和无人机操作员获取空域变化、设施状态等关键信息的主要途径。在 CAAC 考试中，NOTAM 的解读方法和飞行情报的获取流程是高频考点。本节将详细介绍飞行情报服务的体系结构、NOTAM 的格式与解读，以及飞行前情报准备的完整流程。

---

## 1. 飞行情报服务的概念和内容

### 1.1 飞行情报服务的定义

飞行情报服务（Flight Information Service, FIS）是指为保障飞行安全和效率，向航空器提供的有关飞行安全和空中交通管理相关信息的服务。与空中交通管制不同，飞行情报服务不具有强制性指挥权，其核心功能是**信息提供**而非**指令下达**。

### 1.2 飞行情报服务的主要内容

| 情报类型 | 具体内容 | 信息来源 |
| :--- | :--- | :--- |
| **气象情报** | 天气实况、预报、告警 | 气象部门 |
| **航行通告** | 空域变化、设施故障、活动限制 | 航行情报部门 |
| **飞行冲突信息** | 周边航空器的位置和动态 | 管制部门 |
| **导航设施状态** | 导航台、雷达等设施的工作状态 | 设施管理部门 |
| **机场信息** | 跑道状态、灯光系统、地面服务 | 机场管理机构 |

### 1.3 无人机飞行情报的特殊要求

对于无人机运营，飞行情报服务有以下特殊要求：

- **飞行前**：必须获取飞行区域的最新航行通告和气象信息
- **飞行中**：需持续关注空域状态变化和紧急通报
- **飞行后**：记录飞行中收到的情报信息，用于后续分析

## 2. 航行通告（NOTAM）详解

### 2.1 NOTAM 的概念

NOTAM 是 "Notice to Airmen" 的缩写，中文译为"航行通告"。它是由航行情报部门发布的，以电信方式发布的关于航空设施、服务、程序或危险的变更或出现的通告。

### 2.2 NOTAM 的分类

| 类别 | 英文名称 | 适用范围 | 有效期限 |
| :--- | :--- | :--- | :--- |
| **A类 NOTAM** | International NOTAM | 涉及国际飞行的通告 | 通常较长 |
| **B类 NOTAM** | Domestic NOTAM | 仅涉及国内飞行的通告 | 视情况而定 |
| **雪情通告（SNOWTAM）** | SNOWTAM | 机场道面冰雪情况 | 通常较短 |
| **火山灰通告（ASHTAM）** | ASHTAM | 火山灰活动信息 | 视火山活动而定 |

### 2.3 NOTAM 的标准格式

一份标准 NOTAM 由以下部分组成：

**示例 NOTAM：**

```
A) ZBAA
B) 2401150800
C) 2402151800
D) 0001-0600UTC DAILY
E) RWY 36L/18R CLSD DUE MAINT
F) SFC
G) UNL
```

各字段含义解读：

| 字段 | 含义 | 示例解读 |
| :--- | :--- | :--- |
| **NOTAM 编号** | 唯一标识该通告 | 唯一编号 |
| **A）** | 影响范围（ICAO 代码） | ZBAA = 北京首都国际机场 |
| **B）** | 生效日期时间 | 24年1月15日 08:00 UTC |
| **C）** | 失效日期时间 | 24年2月15日 18:00 UTC |
| **D）** | 限制时间段 | 每日 00:01-06:00 UTC |
| **E）** | 通告内容（主体） | 36L/18R 跑道因维护关闭 |
| **F）** | 影响高度下限 | 地面（SFC） |
| **G）** | 影响高度上限 | 无限高（UNL） |

**考试重点**：NOTAM 的 E）字段是理解通告内容的关键，考试中常以 E）字段的内容作为题目素材，要求考生判断对飞行活动的影响。

### 2.4 NOTAM 编码中的关键缩写

| 缩写 | 全称 | 含义 |
| :--- | :--- | :--- |
| RWY | Runway | 跑道 |
| CLSD | Closed | 关闭 |
| DUE | Due to | 由于 |
| MAINT | Maintenance | 维护 |
| SFC | Surface | 地面 |
| UNL | Unlimited | 无限 |
| DLY | Daily | 每日 |
| OPR | Operate | 运行 |
| U/S | Unserviceable | 不可用 |
| ACT | Active | 生效/活动 |
| EXP | Expect | 预计 |
| OBST | Obstacle | 障碍物 |
| LGT | Light | 灯光 |
| APCH | Approach | 进近 |

## 3. 航行情报的获取途径

### 3.1 主要获取渠道

| 渠道 | 方式 | 适用场景 |
| :--- | :--- | :--- |
| **航行情报服务中心** | 官方网站/系统查询 | 飞行前全面准备 |
| **飞行服务站** | 电话/网络查询 | 飞行前及飞行中 |
| **ATC 通报** | 无线电通信 | 飞行中实时获取 |
| **NOTAM 检索系统** | 在线数据库查询 | 按机场/区域筛选 |
| **气象部门** | 专项气象服务 | 气象情报获取 |

### 3.2 无人机运营中的情报获取

无人机运营人需建立系统的情报获取机制：

- **飞行前**：通过航行情报系统查询飞行区域所有有效的 NOTAM
- **飞行中**：保持与飞行服务站或 ATC 的通信联络
- **紧急情况**：通过应急频率获取最新的空域和设施状态信息

## 4. 飞行前情报准备

### 4.1 Pre-flight Briefing 的内容

飞行前情报准备（Pre-flight Information Briefing）是每次飞行前必须完成的工作，内容包括：

| 准备项目 | 具体内容 | 重要性 |
| :--- | :--- | :--- |
| **气象情报** | 天气实况、预报、SIGMET/AIRMET | ★★★★★ |
| **航行通告** | 飞行区域内所有有效 NOTAM | ★★★★★ |
| **航路信息** | 航路结构、导航设施状态 | ★★★★☆ |
| **机场信息** | 起降机场设施状态 | ★★★★☆ |
| **限制区域** | 临时限制区、活动区 | ★★★★★ |
| **紧急程序** | 备降方案、应急频率 | ★★★★☆ |

### 4.2 无人机飞行前检查要点

- 确认飞行区域无临时禁飞限制
- 核实气象条件满足无人机运行限制
- 获取并理解所有相关 NOTAM
- 确认通信频率和应急联络方式
- 检查地面控制站与无人机的通信状态

## 5. 气象情报的种类和获取方式

### 5.1 常用气象情报类型

| 情报类型 | 英文缩写 | 内容 | 更新频率 |
| :--- | :--- | :--- | :--- |
| **机场天气实况** | METAR | 当前天气观测报告 | 每小时 |
| **机场天气预报** | TAF | 未来天气预报 | 每 6-12 小时 |
| **重要天气警告** | SIGMET | 重要气象现象告警 | 按需发布 |
| **区域天气警报** | AIRMET | 区域气象条件变化 | 按需发布 |
| **风切变警告** | WINDSHEAR | 风切变信息 | 按需发布 |
| **特别天气报告** | SPECI | 突变天气的特别报告 | 按需发布 |

### 5.2 METAR 报告解读要点

METAR 是最基础也最重要的气象情报格式，关键参数包括：

- **风**：风向和风速（如 `18015KT` 表示南风 15 节）
- **能见度**：以米为单位的主导能见度
- **天气现象**：降水、雾、霾等（如 `RA` 雨、`FG` 雾）
- **云底高**：云层底部距地面的高度
- **温度/露点**：当前气温和露点温度
- **气压**：修正海平面气压（QNH）

**考试易错点**：METAR 中的风速单位为"节"（KT），而飞行高度以"米"为单位，两种单位制的区分是考试常见陷阱。

## 6. 考试重点总结

### 6.1 NOTAM 解读方法

1. **先看 A）字段**：确认影响的机场或区域
2. **再看 B）和 C）字段**：确认生效和失效时间
3. **重点阅读 E）字段**：理解通告的具体内容
4. **检查 F）和 G）字段**：确认影响的高度范围
5. **综合判断**：评估对本次飞行的具体影响

### 6.2 高频考点归纳

| 考点 | 要求掌握程度 |
| :--- | :--- |
| NOTAM 的 E）字段解读 | 必须熟练掌握 |
| METAR 关键参数识别 | 必须熟练掌握 |
| SIGMET 与 AIRMET 的区别 | 了解并区分 |
| 飞行情报准备的内容清单 | 熟记所有项目 |
| NOTAM 中常见缩写 | 必须全部掌握 |
| 飞行情报获取渠道 | 了解各渠道的适用场景 |

### 6.3 常见考试陷阱

- **时间陷阱**：NOTAM 中的时间均为 **UTC（世界协调时）**，而非北京时间。北京时间 = UTC + 8
- **高度陷阱**：F）和 G）字段中的高度单位需注意区分米和英尺
- **缩写陷阱**：NOTAM 中大量使用缩写，不熟悉缩写可能导致误解通告内容
- **范围陷阱**：NOTAM 的影响范围需准确判断，不要过度扩大或缩小影响范围

掌握飞行情报与航行通告的知识体系，是无人机安全运行的基本功。建议考生在备考时多练习实际 NOTAM 的解读，并养成飞行前进行完整情报准备的良好习惯。
