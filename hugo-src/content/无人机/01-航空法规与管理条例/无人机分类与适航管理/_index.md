# 无人机分类与适航管理

无人机的分类体系是理解整套航空法规的基石。CAAC 根据不同的管理维度——按用途分类、按最大起飞重量分类、按运行场景分类——构建了多层次的分类框架。与之配套的适航管理体系，借鉴了有人航空器的"三证"制度，为无人机的全生命周期管理提供了制度保障。本文将系统梳理无人机分类与适航管理的核心知识体系，重点覆盖考试中的高频混淆点。

---

## 1. 按用途分类

### 1.1 特殊场景无人机（U 系列）

特殊场景无人机指用于特定高风险任务的无人机系统，主要包括：

- **军事用途无人机**：侦察、打击、通信中继等
- **海关缉私用途**：边境巡逻、走私监测
- **警用执法用途**：治安巡逻、交通监控、应急救援
- **应急救援用途**：地震救灾、森林火灾监测

特殊场景无人机通常需要满足更严格的适航和运行标准，其运行合格证以"**U**"开头（如 U-001）。

### 1.2 一般场景无人机（G 系列）

一般场景无人机涵盖所有非特殊场景的民用无人机应用：

- **农业植保**：农药喷洒、作物监测
- **航拍摄影**：影视制作、旅游航拍
- **物流配送**：短途快递、医疗物资运输
- **测绘勘察**：地形测绘、工程勘察
- **巡检维护**：电力线路巡检、风电巡检

一般场景无人机的运行合格证以"**G**"开头（如 G-001），申请流程相对简化。

### 1.3 两类运行合格证对比

| 对比维度 | U 系列（特殊场景） | G 系列（一般场景） |
| :--- | :--- | :--- |
| **适用对象** | 军事、警用、海关等特殊用途 | 农业、航拍、物流等民用用途 |
| **合格证编号格式** | U-XXX | G-XXX |
| **审批机构** | 相关主管部门+民航局协同 | 民航局或地区管理局 |
| **适航要求** | 更严格，参照军用标准 | 按民用标准执行 |
| **运行限制** | 根据任务特殊性定制 | 一般遵循通用运行规则 |

## 2. 按最大起飞重量分类（MTOW）

这是 CAAC 考试中最核心的分类维度。按最大起飞重量（Maximum Take-Off Weight, MTOW）分类，直接决定了无人机的登记要求、适航标准和运行管理方式：

| 类别 | MTOW 范围 | 典型代表 | 管理强度 |
| :--- | :--- | :--- | :--- |
| **微型** | ≤ 250g | 大疆 Mini 系列 | 最低 |
| **轻型** | ＞250g 且 ≤ 4kg | 大疆 Air/Mavic 系列 | 较低 |
| **小型** | ＞4kg 且 ≤ 15kg | 大疆 Matrice/经纬系列 | 中等 |
| **中型** | ＞15kg 且 ≤ 150kg | 工业级固定翼无人机 | 较高 |
| **大型** | ＞150kg | 货运/载人无人机 | 最高 |

### 2.1 各类别的详细管理要求

#### 微型无人机（≤ 250g）

- 无需实名登记
- 无需适航证
- 操控员无需执照
- 仅可在适飞空域内飞行，真高不超过 120m
- **考试重点**：微型是唯一不需要实名登记的类别

#### 轻型无人机（＞250g 且 ≤ 4kg）

- **必须实名登记**，在 UOM 系统完成登记并激活
- 须具备有效的电子围栏功能
- 操控员建议持有执照（非强制要求，但部分地区有附加规定）
- 适飞空域内飞行，真高不超过 120m
- 须投保责任险（部分场景要求）

#### 小型无人机（＞4kg 且 ≤ 15kg）

- 须实名登记
- 须取得适航相关证件
- 操控员须持有相应等级执照
- 须申请飞行空域并获批
- 须提交飞行计划

#### 中型无人机（＞15kg 且 ≤ 150kg）

- 须实名登记
- 须取得完整的适航证（型号合格证+生产许可证+单机适航证）
- 操控员须持有高级别执照
- 严格的空域审批和飞行计划管理
- **额外要求**：须安装广播式自主识别设备（Remote ID）

#### 大型无人机（＞150kg）

- 管理标准向有人航空器看齐
- 须取得完整的三证（TC/PC/AC）
- 须配备专业操控团队
- 按航线运行，须预先申请空域和航线

## 3. 适航证体系（三证制度）

CAAC 无人机适航管理借鉴了有人航空器的成熟体系，建立了"**三证**"制度。理解三证的逻辑关系是考试的重点内容：

### 3.1 型号合格证（Type Certificate, TC）

- **定义**：证明无人机的**设计**符合相关适航标准和安全要求
- **申请人**：无人机设计/制造单位
- **核心内容**：设计方案审查、性能测试、安全评估
- **有效期**：长期有效（设计未变更则持续有效）
- **考点**：TC 针对的是"型号/设计"，而非单架飞机

### 3.2 生产许可证（Production License, PL）

- **定义**：证明制造商具备按照批准的设计**批量生产**无人机的能力
- **申请人**：无人机制造企业
- **核心内容**：生产能力审核、质量管理体系审查、生产一致性检验
- **前提条件**：已取得该型号的 TC
- **考点**：PL 针对的是"生产能力/制造"，而非单架产品

### 3.3 适航证（Airworthiness Certificate, AC）

- **定义**：证明某一架特定的无人机处于**安全可用**状态
- **申请人**：无人机所有人/运营人
- **核心内容**：单机检查、状态确认
- **有效期**：随无人机状态变化，需要持续维护
- **考点**：AC 针对的是"单架飞机"，类似于汽车的行驶证

### 3.4 三证逻辑关系图

```text
型号合格证（TC）         生产许可证（PL）         适航证（AC）
  ↓ 设计合格              ↓ 制造合格               ↓ 单机合格
  └─────────────────────────┴───────────────────────┘
                     ↓
              无人机可合法投入运行
```

| 证书类型 | 关注对象 | 有效期 | 可否转让 |
| :--- | :--- | :--- | :--- |
| **TC（型号合格证）** | 设计/型号 | 长期（设计不变则有效） | 随设计转让 |
| **PL（生产许可证）** | 生产企业 | 审核周期内有效 | 不可转让（企业资质） |
| **AC（适航证）** | 单架无人机 | 持续有效（须维护） | 随飞机转让 |

## 4. 操控员执照等级体系

### 4.1 三级执照架构

CAAC 无人机操控员执照分为三个等级，每个等级对应不同的能力和权限：

| 执照等级 | 英文缩写 | 视距要求 | 典型应用场景 | 权限等级 |
| :--- | :--- | :--- | :--- | :--- |
| **视距内驾驶员** | VLOS | 目视范围内（≤ 500m） | 航拍、小型巡检 | 初级 |
| **超视距驾驶员** | BVLOS | 超出目视范围 | 长航线测绘、物流配送 | 中级 |
| **教员** | Instructor | 视距内外均可 | 培训教学 | 最高 |

### 4.2 各等级的详细要求

#### 视距内驾驶员（VLOS）

- **定义**：操控员在目视范围内直接观察无人机飞行状态
- **飞行距离限制**：水平距离不超过 500m，真高不超过 120m
- **考试科目**：理论考试 + 综合问答 + 实操飞行
- **合格标准**：理论考试 ≥ 70 分
- **适用无人机类别**：微型、轻型、小型

#### 超视距驾驶员（BVLOS）

- **定义**：操控员通过仪表或地面站监控无人机飞行状态
- **飞行距离**：不受视距限制，按审批空域执行
- **额外技能**：须掌握地面站操作、航线规划、应急处置
- **考试科目**：理论考试 + 综合问答 + 实操飞行 + 地面站操作
- **合格标准**：理论考试 ≥ 80 分
- **适用无人机类别**：小型、中型、大型

#### 教员（Instructor）

- **定义**：具备培训和考核其他操控员资质的高级操控员
- **前提条件**：须先取得超视距驾驶员执照，并满足飞行小时数要求
- **考试科目**：理论考试 + 综合问答 + 实操飞行 + 教学法考核
- **合格标准**：理论考试 ≥ 80 分
- **特殊权限**：可在飞行培训中担任教员角色

### 4.3 等级对比表

| 对比维度 | 视距内驾驶员 | 超视距驾驶员 | 教员 |
| :--- | :--- | :--- | :--- |
| **理论合格线** | 70 分 | 80 分 | 80 分 |
| **飞行距离** | ≤ 500m | 无限制 | 无限制 |
| **是否需要地面站** | 否 | 是 | 是 |
| **可否培训他人** | 否 | 否 | 是 |
| **前置条件** | 无 | 须有 VLOS 执照 | 须有 BVLOS 执照 |
| **飞行记录要求** | 基础 | 较多 | 最多 |

## 5. 考试常见分类混淆点

### 5.1 易混淆概念辨析

| 混淆点 | 正确理解 |
| :--- | :--- |
| **微型无人机不需要登记吗** | 是的，微型（≤ 250g）是唯一不需要实名登记的类别 |
| **轻型无人机需要执照吗** | 法规层面不强制，但建议持有；部分地方政府有附加要求 |
| **适航证和登记证是一回事吗** | 不是。登记是信息备案，适航是安全状态确认 |
| **TC 和 AC 有何区别** | TC 管设计型号，AC 管单架飞机 |
| **VLOS 和 BVLOS 的界限** | 以"目视范围"为界限，约 500m 水平距离 |
| **MTOW 包含什么** | 最大起飞重量包括机体、载荷、燃料/电池的总重量 |

### 5.2 考试常见陷阱题型

- **陷阱一**："某无人机最大起飞重量为 248g，请问是否需要实名登记？"——答案是**不需要**，因为 248g ≤ 250g，属于微型无人机。
- **陷阱二**："视距内驾驶员理论考试的合格分数线是多少？"——注意是 **70 分**，不是 80 分。80 分是超视距和教员的合格线。
- **陷阱三**："型号合格证（TC）适用于单架无人机吗？"——不适用，TC 适用于**型号/设计**，单架飞机对应的是适航证（AC）。

## 6. 适航管理的持续要求

取得适航证并非一劳永逸，运营人须持续满足以下要求：

- **定期检查**：按规定的间隔对无人机进行适航性检查
- **维修记录**：完整记录所有维修、改装和零部件更换
- **适航指令**：及时执行 CAAC 发布的适航指令（AD）
- **变更报告**：无人机发生重大改装或事故后须重新评估适航状态
- **证书记载**：适航证上须记载所有影响适航状态的重要信息

## 7. 备考策略建议

- **三证记忆口诀**："TC 管设计、PL 管生产、AC 管单机"——抓住每张证书的核心关注对象
- **分类树形图**：画出五类无人机的分类树，在每个节点标注 MTOW 临界值
- **等级阶梯图**：将三个执照等级画成阶梯，理解"前置条件递进"的逻辑
- **错题归纳法**：专门整理"看似简单实则易错"的分类题，如 250g 临界值、70 分 vs 80 分等
