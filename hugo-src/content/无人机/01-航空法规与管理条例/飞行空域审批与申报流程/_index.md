# 飞行空域审批与申报流程

无人机飞行不是"想飞就飞"，空域审批与飞行计划申报是合法飞行的关键环节。不同类型的无人机、不同的飞行场景，对应的审批流程差异很大。本文将系统梳理空域申请的基本原则、UOM 平台操作流程、各类审批场景的差异，以及考试中常见的"审批 vs 免审批"陷阱，帮助考生建立完整的空域管理知识框架。

---

## 1. 空域申请的基本原则

### 1.1 空域使用的总体框架

无人机空域管理遵循"**分类管理、逐级审批**"的基本原则：

- **适飞空域**：由民航局划定并公布的、在满足基本条件情况下可自由飞行的空域
- **管制空域**：需要事先申请并获得批准后方可使用的空域
- **禁飞区域**：任何情况下均不得飞行的区域

### 1.2 核心管理平台：UOM

**UOM（无人驾驶航空器一体化综合监管服务平台）** 是 CAAC 官方的无人机综合管理平台，承担以下核心功能：

| 功能模块 | 具体用途 |
| :--- | :--- |
| **实名登记** | 无人机所有人信息登记、激活、变更、注销 |
| **空域查询** | 查询适飞空域、管制空域、禁飞区域 |
| **飞行计划申报** | 提交飞行计划、获取审批结果 |
| **临时空域申请** | 申请临时使用的管制空域 |
| **执照管理** | 操控员执照的申请、查询、续签 |
| **事件报告** | 飞行事故、异常事件的报告 |

## 2. 各类无人机的飞行管理差异

### 2.1 微型无人机（≤ 250g）——最简管理

- **空域审批**：无需申请，在适飞空域内可自由飞行
- **飞行计划**：无需提交飞行计划
- **登记要求**：无需实名登记
- **执照要求**：无需操控员执照
- **核心限制**：不得在禁飞区域内飞行，真高不超过 120m

### 2.2 轻型无人机（＞250g 且 ≤ 4kg）——简化管理

- **适飞空域内**：无需空域审批，无需提交飞行计划
- **管制空域内**：须申请空域使用许可并获批
- **飞行限制**：真高不超过 120m，须在白天飞行（夜间须开启防撞灯）
- **额外要求**：须实名登记，须具备电子围栏功能

### 2.3 小型及以上无人机（＞4kg）——严格管理

- **所有飞行**：须申请空域使用许可
- **飞行计划**：须提前提交飞行计划并获批
- **空域申请**：通过 UOM 平台提交，须说明飞行区域、高度、时间、任务等
- **审批时限**：一般提前 1-7 个工作日申请（各地略有差异）

## 3. 临时空域申请流程

### 3.1 申请条件

需要在管制空域内临时飞行的，须满足以下条件：

- 申请人须持有有效的操控员执照
- 无人机已完成实名登记
- 具备相应的适航证件
- 飞行活动不危害公共安全
- 已投保相应的责任保险

### 3.2 申请流程

| 步骤 | 操作内容 | 注意事项 |
| :--- | :--- | :--- |
| **第一步** | 登录 UOM 平台 | 使用实名认证账号登录 |
| **第二步** | 选择"临时空域申请" | 填写申请表格 |
| **第三步** | 填写空域参数 | 空域范围、高度层、使用时段 |
| **第四步** | 提交任务说明 | 任务性质、飞行区域地图、应急预案 |
| **第五步** | 等待审批 | 审批时限一般为 1-5 个工作日 |
| **第六步** | 获取批复 | 获批后方可按计划飞行 |

### 3.3 审批时限与使用期限

| 参数 | 具体要求 |
| :--- | :--- |
| **审批时限** | 一般为 1-5 个工作日（紧急情况可加急处理） |
| **使用期限** | 临时空域使用期限最长不超过 **12 个月** |
| **续期申请** | 需要延期使用的，应在到期前重新申请 |
| **变更申请** | 飞行计划有重大变更的，须重新申请 |

## 4. 飞行计划申报要素

### 4.1 申报必须包含的信息

一次完整的飞行计划申报应包含以下核心要素：

| 申报要素 | 具体内容 | 考试考点 |
| :--- | :--- | :--- |
| **飞行区域** | 起飞点、飞行范围、降落点的地理坐标 | 须精确到经纬度 |
| **飞行高度** | 计划飞行的真高（AGL） | 注意是真高不是海拔 |
| **飞行时间** | 起止时间、预计持续时长 | 须在获批时段内飞行 |
| **任务性质** | 航拍、测绘、巡检、配送等 | 须如实填写 |
| **操控员信息** | 姓名、执照编号、联系方式 | 须为持证操控员 |
| **无人机信息** | 型号、登记编码、适航状态 | 须在有效期内 |
| **应急预案** | 失联处置方案、紧急迫降区域 | 须预先规划 |

### 4.2 飞行计划的变更与取消

- **计划变更**：飞行计划有重大变更时，须提前向审批机关报告并重新申请
- **计划取消**：因天气等原因无法按计划飞行的，须及时取消飞行计划
- **实际偏差**：实际飞行中与计划有偏差的，应在飞行结束后及时报告
- **考点提示**："飞行结束后及时报告"是常考的程序性要求

## 5. 禁飞区域详解

### 5.1 绝对禁飞区域

| 禁飞区域 | 具体范围 | 禁飞原因 |
| :--- | :--- | :--- |
| **机场净空区** | 跑道中心线两侧各 10km，跑道端外 20km | 保障民航飞行安全 |
| **军事管制区** | 军事禁区、军事管理区及其周边 | 国防安全 |
| **核设施区域** | 核电站、核废料处理设施周边 | 核安全 |
| **政府重要机关** | 党政机关办公区域 | 政治安全 |
| **监管场所** | 监狱、看守所等 | 监管安全 |
| **大型水利设施** | 大型水库大坝及其周边 | 水利安全 |

### 5.2 限飞区域

| 限飞区域 | 限飞要求 | 审批条件 |
| :--- | :--- | :--- |
| **真高 120m 以上** | 所有无人机须审批 | 按空域管理规定申请 |
| **管制空域** | 须申请空域使用许可 | 提前申请并获批 |
| **人口密集区** | 特殊审批 | 须满足安全条件 |
| **临时管制区** | 临时禁飞或限飞 | 按通告执行 |

### 5.3 净空区参数速记

机场净空区的参数是考试高频考点：

- 跑道中心线两侧：各 **10km**
- 跑道端外：**20km**
- 净空区内禁止一切可能影响飞行安全的活动
- **考试陷阱**：注意是"两侧各 10km"，不是"两侧共 10km"

## 6. 考试陷阱：审批 vs 免审批

### 6.1 无需审批的情形

| 情形 | 条件 | 关键限制 |
| :--- | :--- | :--- |
| **微型无人机在适飞空域** | ≤ 250g，适飞空域内 | 真高 ≤ 120m |
| **轻型无人机在适飞空域** | ≤ 4kg，适飞空域内 | 真高 ≤ 120m，白天飞行 |

### 6.2 须审批的情形

| 情形 | 需要的审批 | 审批渠道 |
| :--- | :--- | :--- |
| **任何无人机在管制空域飞行** | 空域使用许可 | UOM 平台 |
| **小型及以上无人机的所有飞行** | 空域许可+飞行计划 | UOM 平台 |
| **夜间飞行（特定类型）** | 夜间飞行许可 | UOM 平台 |
| **人口密集区飞行** | 特殊审批 | 地方管理局 |
| **真高 120m 以上飞行** | 空域使用许可 | UOM 平台 |

### 6.3 考试常见陷阱题

- **陷阱一**："轻型无人机在适飞空域内飞行需要申请飞行计划吗？"——**不需要**，适飞空域内轻型无人机免审批
- **陷阱二**："微型无人机可以在夜间飞行吗？"——**不可以**，微型无人机不允许夜间飞行
- **陷阱三**："在机场净空区内飞行小型无人机，审批能否通过？"——**不能**，净空区是绝对禁飞区，无论何种审批都无法获批
- **陷阱四**："无人机飞行高度 110m，是否需要审批？"——取决于无人机类型和区域，轻型在适飞空域内 110m < 120m，**不需要**审批
- **陷阱五**："超视距飞行一定需要审批吗？"——是的，超视距飞行意味着超出目视范围，**通常需要**相应的空域许可和资质

## 7. 空域管理相关数字速记

| 数字 | 含义 | 记忆方法 |
| :--- | :--- | :--- |
| **120m** | 适飞空域飞行高度上限（真高） | "120"是一条红线 |
| **500m** | 视距内驾驶员的水平操控距离上限 | "500m 目视" |
| **10km** | 机场净空区跑道中心线两侧宽度 | "两侧各10" |
| **20km** | 机场净空区跑道端外延伸距离 | "端外20" |
| **12个月** | 临时空域最长使用期限 | "一年一批" |
| **1-5个工作日** | 飞行计划审批时限 | "提前申请" |
| **30日** | 飞行计划变更/取消后的报告时限 | "30天内报告" |

## 8. 特殊场景的空域管理

### 8.1 夜间飞行

- 微型无人机：**不允许**夜间飞行
- 轻型无人机：须开启防撞灯，部分区域仍需审批
- 小型及以上：须获得专项审批

### 8.2 人口密集区飞行

- 微型和轻型无人机：**禁止**在人口密集区域上方飞行
- 小型及以上：须获得特殊审批，满足安全冗余要求
- **考点**：人口密集区包括城镇市区、集市、车站、大型活动现场等

### 8.3 跨区域飞行

- 跨越不同管制区域的飞行，须分别获得各区域的空域许可
- 跨省飞行须向相关地区管理局报告
- **考点**：跨区域飞行的审批可能涉及多个空域管理机构

## 9. 备考建议

- **场景判断法**：拿到题目先判断无人机类型（微型/轻型/小型...），再判断飞行区域（适飞/管制/禁飞），最后得出是否需要审批
- **数字联想法**：将 120m、500m、10km、20km 等数字与具体场景关联记忆
- **逆向排除法**：遇到"是否需要审批"的题目，先列出所有免审批情形，不在列表中的即需要审批
- **流程图记忆法**：画出临时空域申请的完整流程图，标注每个步骤的关键操作
