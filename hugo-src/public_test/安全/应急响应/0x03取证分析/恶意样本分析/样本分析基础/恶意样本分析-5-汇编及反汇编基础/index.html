2. CPUеҜ„еӯҳеҷЁ

CPUеҢ…еҗ«зү№ж®Ҡзҡ„еӯҳеӮЁжҲҗдёәеҜ„еӯҳеҷЁгҖӮ CPUи®ҝй—®еҜ„еӯҳеҷЁйҮҢзҡ„ж•°жҚ®жҜ”и®ҝй—®еҶ…еӯҳдёӯзҡ„ж•°жҚ®иҰҒеҝ«зҡ„еӨҡгҖӮеӣ дёәеҶ…еӯҳдёӯзҡ„ж•°жҚ®иҰҒе…ҲжӢҝеҲ°еҜ„еӯҳеҷЁдёӯеҶҚиў«CPUжү§иЎҢгҖӮ

2.1 йҖҡз”ЁеҜ„еӯҳеҷЁ

x86CPUжңү8дёӘйҖҡз”ЁеҜ„еӯҳеҷЁпјҡeax,ebx,ecx,edx,esp,ebp,esi,ediгҖӮиҝҷдәӣеҜ„еӯҳеҷЁжҳҜ32дҪҚпјҲ4еӯ—иҠӮпјүгҖӮзЁӢеәҸеҸҜд»ҘиҺ·еҸ–еҜ„еӯҳеҷЁ32дҪҚпјҢ16дҪҚпјҢ8дҪҚеҖјгҖӮжҜҸдёӘеҜ„еӯҳеҷЁзҡ„дҪҺ16дҪҚпјҲ2еӯ—иҠӮпјүеҸҜд»Ҙз”Ёax,bx,cx,dx,sp,bp,si,diи®ҝй—®гҖӮeax.ebx,ecx,edxзҡ„дҪҺ8дҪҚиҝҳеҸҜд»ҘйҖҡиҝҮal,bl,cl,dlеј•з”ЁгҖӮеҜ№еә”зҡ„й«ҳ8дҪҚеҸҜд»ҘйҖҡиҝҮah,bh,ch,dhи®ҝй—®гҖӮдёҫдҫӢеҰӮдёӢеӣҫжүҖзӨәпјҡ

2.2 жҢҮд»ӨжҢҮй’ҲпјҲEIPпјү

CPUеӯҳеңЁдёҖдёӘзү№ж®Ҡзҡ„еҜ„еӯҳеҷЁeipпјӣе®ғеҢ…еҗ«дёӢдёҖдёӘиҰҒжү§иЎҢзҡ„зҡ„жҢҮд»Өзҡ„ең°еқҖгҖӮеҪ“жҢҮд»Өиў«жү§иЎҢпјҢeipе°ҶдјҡжҢҮеҗ‘еҶ…еӯҳдёӯдёӢдёҖдёӘе°Ҷиў«жү§иЎҢзҡ„жҢҮд»Өең°еқҖгҖӮ

2.3 EFLAGSеҜ„еӯҳеҷЁ

eflagsеҜ„еӯҳеҷЁжҳҜ32дҪҚеҜ„еӯҳеҷЁпјҢиҜҘеҜ„еӯҳеҷЁзҡ„жҜҸдёҖдҪҚйғҪд»ЈиЎЁдёҖдёӘзү№ж®Ҡеҗ«д№үзҡ„ж Үи®°гҖӮeflagsдёӯзҡ„дҪҚдҪҝз”ЁжқҘд»ЈиЎЁCPUиҝҗз®—дёӯи®Ўз®—жҲ–жҺ§еҲ¶зҡ„зҠ¶жҖҒзҡ„гҖӮflagеҜ„еӯҳеҷЁйҖҡеёёдёҚзӣҙжҺҘеј•з”ЁпјҢдҪҶжҳҜеңЁжү§иЎҢи®Ўз®—жҲ–жҺ§еҲ¶ж—¶пјҢжҜҸдёҖдҪҚдјҡж №жҚ®з»“жһңиҝӣиЎҢеҸҳеҢ–гҖӮ

йҷӨжӯӨд№ӢеӨ–иҝҳжңүдёҖдәӣйўқеӨ–зҡ„еҜ„еӯҳеҷЁиў«з§°дёәж®өеҜ„еӯҳеҷЁпјҡcs,ss,ds,es,fs,gsпјҢиў«з”ЁжқҘеңЁеҶ…еӯҳдёӯдҝқжҢҒиҝҪиёӘзҡ„гҖӮ

3.3 з§»еҠЁеҖјд»ҺеҶ…еӯҳеҲ°еҜ„еӯҳеҷЁ

int val=100 еңЁзЁӢеәҸжү§иЎҢж—¶еҸ‘з”ҹзҡ„жғ…еҶөпјҡ

  1. ж•ҙж•°й•ҝеәҰдёә4еӯ—иҠӮпјҢеӣ жӯӨж•ҙж•°100еңЁеҶ…еӯҳдёӯиў«еӯҳеӮЁдёәпјҲ00 00 00 64пјү
  2. 4еӯ—иҠӮеәҸеҲ—иў«жҢүз…§дҪҺдҪҚдјҳе…Ҳж јејҸеӯҳж”ҫ
  3. ж•ҙж•°100иў«еӯҳеӮЁеңЁзӣёеҗҢзҡ„еҶ…еӯҳең°еқҖдёӢгҖӮ

еңЁжұҮзј–иҜӯиЁҖдёӯпјҢз§»еҠЁеҶ…еӯҳдёӯзҡ„еҖјеҲ°еҜ„еӯҳеҷЁдёӯпјҢеҝ…йЎ»иҰҒдҪҝз”ЁеҖјзҡ„ең°еқҖгҖӮ ж–№жӢ¬еҸ·жҢҮе®ҡзҡ„ж—¶иҰҒдҝқеӯҳзҡ„еҖјеңЁеҶ…еӯҳдёӯзҡ„ең°еқҖгҖӮ

mov eax,[0x403000] ; eax will now contain 00 00 00 64 (i.e 100)

иҝҷйҮҢж— йңҖжҢҮе®ҡ4еӯ—иҠӮпјҢеҹәдәҺзӣ®ж ҮеҜ„еӯҳеҷЁзҡ„еӨ§е°ҸпјҢCPUдјҡиҮӘеҠЁзЎ®и®ӨйңҖиҰҒз§»еҠЁеӨҡе°‘еӯ—иҠӮгҖӮ

йҖҶеҗ‘иҝҮзЁӢдёӯзҡ„е…¶д»–зұ»еһӢиҝҳжңүеҰӮпјҢж–№жӢ¬еҸ·еҢ…еҗ«еҜ„еӯҳеҷЁгҖҒеёёж•°+еҜ„еӯҳеҷЁгҖҒеҜ„еӯҳеҷЁ+еҜ„еӯҳеҷЁзҡ„еҪўејҸгҖӮ

mov eax,[ebx]     ; moves value at address specifed by ebx register
mov eax,[ebx+ecx] ; moves value at address specified by ebx+ecx

3.4 з§»еҠЁеҖјд»ҺеҜ„еӯҳеҷЁеҲ°еҶ…еӯҳ

дҪ йҖҡиҝҮз§»еҠЁж“ҚдҪңж•°еҸҜд»Ҙз§»еҠЁдёҖдёӘеҖјд»ҺдёҖдёӘеҜ„еӯҳеҷЁеҲ°еҶ…еӯҳпјҢеҶ…еӯҳең°еқҖеңЁзӣ®ж ҮдҪҚзҪ®еңЁе·Ұиҫ№пјҢеҜ„еӯҳеҷЁеңЁеҸіиҫ№гҖӮ

3.6 еҸҚжұҮзј–и§Јжі•

з®ҖеҚ•зҡ„ж–№ејҸзҗҶи§ЈеҸҚжұҮзј–д»Јз ҒпјҢеңЁеҜ№жҜ”cиҜӯиЁҖдёӯпјҢдёҖдёӘе®ҡд№үзҡ„еҸҳйҮҸе®һйҷ…дёҠе°ұжҳҜдёҖдёӘеҶ…еӯҳең°еқҖзҡ„иұЎеҫҒеҗҚеӯ—гҖӮз»ҸиҝҮйҖ»иҫ‘пјҢз„¶еҗҺеҸҜд»Ҙе®ҡд№үеҶ…еӯҳең°еқҖз»ҷд»–д»¬дёҖдёӘиұЎеҫҒзҡ„еҗҚеӯ—гҖӮ

еҶ…еӯҳең°еқҖпјҢзӣҙжҺҘз»ҷдёҖдёӘж Үи®°еҗҚеӯ—пјҢеҰӮ[ebp-4]=a,[ebp-8]=b

say,В ebp-4 = a and ebp-8 = b. Now, the program should look like the one shown here: mov dword ptr [a],1     ; treat it as mov [a],1
mov eax,dword ptr [a]   ; treat it as mov eax,[a]
mov dword ptr [b],eax   ; treat it as mov [b],eax

з”ұдәҺCPUдҪҝз”ЁеҜ„еӯҳеҷЁжҡӮеӯҳпјҢеӣ жӯӨиҝҳйңҖиҰҒдҪҝз”ЁеҸіиҫ№зҡ„ж Үи®°еҖјжӣҝжҚўеҜ„еӯҳеҷЁзҡ„еҗҚеӯ—пјҢдҫӢеҰӮeaxдҪҝз”ЁaжӣҝжҚў

еҜ„еӯҳеҷЁдҪҝз”ЁеҸіиҫ№зҡ„еӨҚеҲ¶ж Үи®°еҖјжӣҝжҚў

иҖҢaпјҢbдёҚеҸҜиғҪжҳҜpointerжҢҮй’Ҳзҡ„еҺҹеӣ жҳҜпјҢд»–д»¬иөӢеҖјдёә1пјҢдёҖдёӘеёёж•°пјҢиҖҢдёҚжҳҜдёҖдёӘең°еқҖпјҢеӣ жӯӨжңҖз»ҲзЎ®е®ҡжҳҜж•ҙж•°зұ»еһӢгҖӮ
