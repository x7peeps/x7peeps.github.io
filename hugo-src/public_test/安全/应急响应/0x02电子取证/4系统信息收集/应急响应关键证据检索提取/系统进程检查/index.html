Linuxзі»з»ҹдёҠзҡ„/procзӣ®еҪ•жҳҜдёҖз§Қж–Үд»¶зі»з»ҹпјҢеҚіprocж–Үд»¶зі»з»ҹгҖӮ

е…¶е®ғеёёи§Ғзҡ„ж–Үд»¶зі»з»ҹдёҚеҗҢзҡ„жҳҜпјҢ/procжҳҜдёҖз§ҚдјӘж–Үд»¶зі»з»ҹпјҲд№ҹеҚіиҷҡжӢҹж–Үд»¶зі»з»ҹпјүпјҢеӯҳеӮЁзҡ„жҳҜеҪ“еүҚеҶ…ж ёиҝҗиЎҢзҠ¶жҖҒзҡ„дёҖзі»еҲ—зү№ж®Ҡж–Үд»¶пјҢз”ЁжҲ·еҸҜд»ҘйҖҡиҝҮиҝҷдәӣж–Үд»¶жҹҘзңӢжңүе…ізі»з»ҹзЎ¬д»¶еҸҠеҪ“еүҚжӯЈеңЁиҝҗиЎҢиҝӣзЁӢзҡ„дҝЎжҒҜпјҢз”ҡиҮіеҸҜд»ҘйҖҡиҝҮжӣҙж”№е…¶дёӯжҹҗдәӣж–Үд»¶жқҘж”№еҸҳеҶ…ж ёзҡ„иҝҗиЎҢзҠ¶жҖҒгҖӮ

дёәдәҶжҹҘзңӢеҸҠдҪҝз”ЁдёҠзҡ„ж–№дҫҝпјҢиҝҷдәӣж–Үд»¶йҖҡеёёдјҡжҢүз…§зӣёе…іжҖ§иҝӣиЎҢеҲҶзұ»еӯҳеӮЁдәҺдёҚеҗҢзҡ„зӣ®еҪ•з”ҡиҮіеӯҗзӣ®еҪ•дёӯпјҢеҰӮ/proc/scsiзӣ®еҪ•дёӯеӯҳеӮЁзҡ„е°ұжҳҜеҪ“еүҚзі»з»ҹдёҠжүҖжңүSCSIи®ҫеӨҮзҡ„зӣёе…ідҝЎжҒҜпјҢ/proc/NдёӯеӯҳеӮЁзҡ„еҲҷжҳҜзі»з»ҹеҪ“еүҚжӯЈеңЁиҝҗиЎҢзҡ„иҝӣзЁӢзҡ„зӣёе…ідҝЎжҒҜпјҢе…¶дёӯNдёәжӯЈеңЁиҝҗиЎҢзҡ„иҝӣзЁӢпјҲеҸҜд»ҘжғіиұЎеҫ—еҲ°пјҢеңЁжҹҗиҝӣзЁӢз»“жқҹеҗҺе…¶зӣёе…ізӣ®еҪ•еҲҷдјҡж¶ҲеӨұпјүгҖӮ

еӨ§еӨҡж•°иҷҡжӢҹж–Үд»¶еҸҜд»ҘдҪҝз”Ёж–Үд»¶жҹҘзңӢе‘Ҫд»ӨеҰӮcatгҖҒmoreжҲ–иҖ…lessиҝӣиЎҢжҹҘзңӢпјҢжңүдәӣж–Үд»¶дҝЎжҒҜиЎЁиҝ°зҡ„еҶ…е®№еҸҜд»ҘдёҖзӣ®дәҶз„¶пјҢдҪҶд№ҹжңүж–Үд»¶зҡ„дҝЎжҒҜеҚҙдёҚжҖҺд№Ҳе…·жңүеҸҜиҜ»жҖ§гҖӮдёҚиҝҮпјҢиҝҷдәӣеҸҜиҜ»жҖ§иҫғе·®зҡ„ж–Үд»¶еңЁдҪҝз”ЁдёҖдәӣе‘Ҫд»ӨеҰӮapmгҖҒfreeгҖҒlspciжҲ–topжҹҘзңӢж—¶еҚҙеҸҜд»ҘжңүзқҖдёҚй”ҷзҡ„иЎЁзҺ°гҖӮ

жҹҘзңӢеҪ“еүҚзі»з»ҹиҝӣзЁӢ

/procзӣ®еҪ•дёӯеҢ…еҗ«и®ёеӨҡд»Ҙж•°еӯ—е‘ҪеҗҚзҡ„еӯҗзӣ®еҪ•пјҢиҝҷдәӣж•°еӯ—иЎЁзӨәзі»з»ҹеҪ“еүҚжӯЈеңЁиҝҗиЎҢиҝӣзЁӢзҡ„иҝӣзЁӢеҸ·пјҢйҮҢйқўеҢ…еҗ«еҜ№еә”иҝӣзЁӢзӣёе…ізҡ„еӨҡдёӘдҝЎжҒҜж–Үд»¶гҖӮ

pstreeпјҲжҜ”иҫғеҘҪз”Ёпјү

зӣёиҫғдәҺ/procжҹҘиҜўпјҢpstreeзҡ„еұ•зӨәжӣҙеҠ ж–№дҫҝпјҢжҜ”psеұ•зӨәзҡ„жӣҙеҠ жё…жҷ°жңүеұӮж¬Ў

pstree [-a] [-c] [-h|-Hpid] [-l] [-n] [-p] [-u] [-G|-U] [pid|user]

-a жҳҫзӨәиҜҘиЎҢзЁӢзҡ„е®Ңж•ҙжҢҮд»ӨеҸҠеҸӮж•°, еҰӮжһңжҳҜиў«и®°еҝҶдҪ“зҪ®жҚўеҮәеҺ»зҡ„иЎҢзЁӢеҲҷдјҡеҠ дёҠжӢ¬еҸ·
-c еҰӮжһңжңүйҮҚиҰҶзҡ„иЎҢзЁӢеҗҚ, еҲҷеҲҶејҖеҲ—еҮәпјҲйў„и®ҫеҖјжҳҜдјҡеңЁеүҚйқўеҠ дёҠ *пјү
-A еҗ„иҝӣзЁӢж ‘д№Ӣй—ҙзҡ„иҝһжҺҘд»ҘASCIIз Ғеӯ—з¬ҰжқҘиҝһжҺҘ
-U еҗ„иҝӣзЁӢж ‘д№Ӣй—ҙзҡ„иҝһжҺҘд»Ҙutf8еӯ—з¬ҰжқҘиҝһжҺҘпјҢжҹҗдәӣз»Ҳз«ҜеҸҜиғҪдјҡжңүй”ҷиҜҜ
-p еҗҢж—¶еҲ—еҮәжҜҸдёӘиҝӣзЁӢзҡ„PID
-u еҗҢж—¶еҲ—еҮәжҜҸдёӘиҝӣзЁӢзҡ„жүҖеұһиҙҰеҸ·еҗҚз§°(иў«е°ҸжӢ¬еҸ·жӢ¬иө·жқҘ)

зү№еҲ«иЎЁжҳҺеңЁиҝҗиЎҢзҡ„иҝӣзЁӢ
# pstree -apnh //жҳҫзӨәиҝӣзЁӢй—ҙзҡ„е…ізі»
еҗҢж—¶жҳҫзӨәз”ЁжҲ·еҗҚз§°

# pstree -u //жҳҫзӨәз”ЁжҲ·еҗҚз§°
