  • е·ІжҺ§еҲ¶зҡ„жңәеҷЁиҙҰжҲ·
  • е·ІжҺ§еҲ¶дё”еёҰжңү SPN зҡ„жңҚеҠЎиҙҰжҲ·

жңҖеёёи§Ғзҡ„жҳҜз¬¬дёҖз§ҚгҖӮи®ёеӨҡеҹҹд»Қдҝқз•ҷй»ҳи®Ө MachineAccountQuota=10пјҢжҷ®йҖҡеҹҹз”ЁжҲ·еҚіеҸҜеҲӣе»әжңәеҷЁиҙҰжҲ·гҖӮжңәеҷЁиҙҰжҲ·еӨ©з„¶е…·еӨҮ HOST/гҖҒCIFS/ зӯү SPNпјҢеҸҜзӣҙжҺҘеҸӮдёҺ S4U2Self е’Ң S4U2Proxy жөҒзЁӢгҖӮ

дҪҶиҝҷйҮҢжңүдёӘе®№жҳ“иў«еҝҪз•Ҙзҡ„иҫ№з•Ңпјҡ

  • жҷ®йҖҡз”ЁжҲ·й»ҳи®ӨжІЎжңү SPN
  • жІЎжңү SPN зҡ„жҷ®йҖҡи·Ҝеҫ„дёӢж— жі•зӣҙжҺҘе®ҢжҲҗеёёи§Ғ S4U2Self
  • жңәеҷЁиҙҰжҲ·д№ӢжүҖд»Ҙеёёз”ЁпјҢдёҚжҳҜвҖңеҝ…йЎ»жҳҜжңәеҷЁвҖқпјҢиҖҢжҳҜеӣ дёәе®ғеӨ©з„¶ж»Ўи¶і SPN жқЎд»¶дё”еҲӣе»әй—Ёж§ӣдҪҺ

1.1.1 зӣ®ж ҮзҺҜеўғеҝ«йҖҹжЈҖжҹҘеҚ•

еңЁжӯЈејҸиө°й“ҫеүҚпјҢе»әи®®зәўйҳҹе…Ҳе®ҢжҲҗдёӢйқўиҝҷз»„жңҖе°ҸеҲӨж–ӯпјҡ

Import-Module .\PowerView.ps1

# еҹҹжҺ§зүҲжң¬/зҺҜеўғеҲӨж–ӯ
Get-DomainController | Select-Object Name, OSVersion

# жңәеҷЁиҙҰжҲ·й…Қйўқ
Get-DomainObject -Identity "DC=corp,DC=local" -Properties ms-ds-machineaccountquota

# зӣ®ж ҮеҜ№иұЎеҪ“еүҚжҳҜеҗҰе·Іжңү RBCD
Get-DomainComputer FILE01 -Properties msds-allowedtoactonbehalfofotheridentity

еҰӮжһңзӣ®ж ҮеҜ№иұЎжң¬жқҘе°ұе·Із»ҸеӯҳеңЁеҗҲжі•е§”жҙҫжқЎзӣ®пјҢеҗҺз»ӯеӨ„зҪ®е’Ңйҡҗи”ҪжҖ§иҖғиҷ‘дјҡе®Ңе…ЁдёҚеҗҢгҖӮзәўйҳҹйңҖиҰҒзЎ®и®ӨиҮӘе·ұжҳҜеңЁиҝҪеҠ пјҢиҝҳжҳҜдјҡиҰҶзӣ–еҺҹжңүй…ҚзҪ®гҖӮ

1.2 зәўйҳҹжңҖе…іеҝғзҡ„жһҡдёҫзӮ№

жӢҝеҲ°дҪҺжқғйҷҗиҙҰжҲ·еҗҺпјҢдјҳе…ҲзңӢдёүзұ»дҝЎжҒҜпјҡ

Import-Module .\PowerView.ps1

# 1. жҹҘ MachineAccountQuota
Get-DomainObject -Identity "DC=corp,DC=local" -Properties ms-ds-machineaccountquota

# 2. жһҡдёҫеҜ№и®Ўз®—жңәеҜ№иұЎзҡ„еҶҷжқғйҷҗ
Get-DomainObjectAcl -ResolveGUIDs |
  Where-Object {
    $_.ActiveDirectoryRights -match "GenericWrite|WriteDacl|GenericAll|WriteProperty"
  } |
  Select-Object ObjectDN, ActiveDirectoryRights, SecurityIdentifier

еҰӮжһңз”Ё BloodHoundпјҢйҮҚзӮ№дёҚжҳҜвҖңи·Ҝеҫ„й•ҝдёҚй•ҝвҖқпјҢиҖҢжҳҜпјҡ

  • еҪ“еүҚз”ЁжҲ·жҳҜеҗҰжӢҘжңү AddComputer
  • еҪ“еүҚз”ЁжҲ·жҲ–е…¶з»„жҳҜеҗҰиғҪеҶҷжҹҗеҸ°жңҚеҠЎеҷЁеҜ№иұЎ
  • иў«еҶҷеҜ№иұЎжҳҜеҗҰжҳҜи·іжқҝжңәгҖҒж–Үд»¶жңҚеҠЎеҷЁгҖҒиҜҒд№ҰжңҚеҠЎеҷЁгҖҒз®ЎзҗҶиҠӮзӮ№жҲ–еҹҹжҺ§

иЎҘе……дёҖзұ»еңЁйЎ№зӣ®йҮҢйқһеёёеёёи§ҒдҪҶз»Ҹеёёиў«еҝҪи§Ҷзҡ„и·Ҝеҫ„пјҡ

  • WriteAccountRestrictions

жҹҗдәӣзҺҜеўғдјҡйў„е…ҲеҲӣе»әи®Ўз®—жңәеҜ№иұЎпјҢеҶҚжҠҠеҗҺз»ӯеҠ еҹҹжҲ–дәӨд»ҳжқғйҷҗдәӨз»ҷйЎ№зӣ®з»„гҖҒжЎҢйқўиҝҗз»ҙжҲ–е®һж–Ҫдәәе‘ҳгҖӮиҝҷдёӘжқғйҷҗдёҚдёҖе®ҡзңӢиө·жқҘеғҸвҖңе…ЁжҺ§вҖқпјҢдҪҶи¶ід»Ҙи§Ұиҫҫе…ій”®еұһжҖ§йӣҶпјҢжңҖз»ҲеҗҢж ·иғҪеҸҳжҲҗ RBCDгҖӮ

1.2.1 е“Әдәӣ ACL зңӢеҲ°е°ұиҜҘз«ӢеҲ»дёҠеҝғ

еҜ№и®Ўз®—жңәеҜ№иұЎжқҘиҜҙпјҢдёӢйқўеҮ зұ»иҫ№еңЁ BloodHound жҲ– ACL жһҡдёҫдёӯйғҪеҖјеҫ—йҮҚзӮ№ж Үзәўпјҡ

  • GenericAll
  • GenericWrite
  • WriteDacl
  • WriteOwner
  • Owns
  • WriteAccountRestrictions
  • AllowedToAct

иҝҷдәӣжқғйҷҗзҡ„ж”»еҮ»еҗ«д№үе№¶дёҚе®Ңе…ЁдёҖж ·пјҡ

  • GenericWrite / GenericAll йҖҡеёёиғҪзӣҙжҺҘиҗҪеҲ°еҶҷеұһжҖ§
  • WriteDacl е…Ҳз»ҷиҮӘе·ұиЎҘжқғйҷҗпјҢеҶҚеҶҷеұһжҖ§
  • WriteOwner е…ҲеӨәеҜ№иұЎжүҖжңүжқғпјҢеҶҚж”№ DACL
  • AllowedToAct еҫҖеҫҖж„Ҹе‘ізқҖзӣ®ж ҮеҜ№иұЎе·Із»ҸеӯҳеңЁзҺ°жҲҗзҡ„ RBCD жқЎзӣ®пјҢз”ҡиҮіеҸҜзӣҙжҺҘеҲ©з”Ё

1.3 зҘЁжҚ®еұӮйқўзңҹжӯЈеҸ‘з”ҹдәҶд»Җд№Ҳ

еҫҲеӨҡж–Үз« жҠҠ RBCD еҶҷжҲҗвҖңдҝ®ж”№еұһжҖ§еҗҺжӢҝз®ЎзҗҶе‘ҳжқғйҷҗвҖқпјҢе®һйҷ…е№¶дёҚеҮҶзЎ®гҖӮRBCD зҡ„дә§еҮәдёҚжҳҜз®ЎзҗҶе‘ҳеҜҶз ҒпјҢиҖҢжҳҜдёҖеј еҗҲжі•зҡ„жңҚеҠЎзҘЁжҚ®гҖӮ

е®Ңж•ҙжөҒзЁӢйҖҡеёёжҳҜпјҡ

  1. ж”»еҮ»иҖ…жҺ§еҲ¶дёҖдёӘжңҚеҠЎдё»дҪ“
  2. жҠҠиҜҘдё»дҪ“еҶҷе…Ҙзӣ®ж ҮеҜ№иұЎзҡ„ msDS-AllowedToActOnBehalfOfOtherIdentity
  3. дҪҝз”ЁиҜҘдё»дҪ“еҗ‘ KDC еҸ‘иө· S4U2Self
  4. еҶҚеҸ‘иө· S4U2Proxy
  5. иҺ·еҫ—йқўеҗ‘зӣ®ж ҮжңҚеҠЎзҡ„ TGS

жңҖз»Ҳж•ҲжһңжҳҜпјҡж”»еҮ»иҖ…дёҚйңҖиҰҒзҹҘйҒ“ Administrator зҡ„еҜҶз ҒпјҢд№ҹеҸҜд»ҘжӢҝеҲ°дёҖеј вҖңд»ЈиЎЁ Administrator и®ҝй—® cifs/file01.corp.localвҖқзҡ„зҘЁжҚ®гҖӮ

иҝҷйҮҢзҡ„е…ій”®дёҚжҳҜвҖңеҶ’е……з®ЎзҗҶе‘ҳжң¬дәәеҒҡжүҖжңүдәӢвҖқпјҢиҖҢжҳҜвҖңжӢҝеҲ°з®ЎзҗҶе‘ҳи®ҝй—®жҹҗдёӘзӣ®ж ҮжңҚеҠЎзҡ„йӮЈдёҖеј зҘЁжҚ®вҖқгҖӮеӣ жӯӨеҲ©з”ЁжҲҗеҠҹеҗҺпјҢж”»еҮ»иҖ…еҗҺз»ӯеҲ°еә•иғҪдёҚиғҪзӣҙжҺҘ psexecгҖҒwmiexec жҲ– secretsdumpпјҢеҸ–еҶідәҺпјҡ

  • зӣ®ж ҮжңҚеҠЎжҳҜеҗҰж”ҜжҢҒеҜ№еә”ж“ҚдҪң
  • иў«жЁЎжӢҹз”ЁжҲ·жҳҜеҗҰеҜ№зӣ®ж Үдё»жңәжңүи¶іеӨҹжқғйҷҗ
  • и®ҝй—®ж—¶жҳҜеҗҰжӯЈзЎ®дҪҝз”Ёдё»жңәеҗҚ/FQDN иҖҢдёҚжҳҜ IP

1.4 йҖүзӣ®ж Үж—¶зҡ„дјҳе…Ҳзә§

еҫҲеӨҡж–°жүӢдёҖзңӢеҲ° RBCD е°ұеҸӘзӣҜеҹҹжҺ§пјҢдҪҶйЎ№зӣ®йҮҢжӣҙй«ҳж•Ҳзҡ„йЎәеәҸйҖҡеёёжҳҜпјҡ

  1. иҝҗз»ҙи·іжқҝжңәгҖҒе Ўеһ’жңәгҖҒз®ЎзҗҶжңә
  2. ж–Үд»¶жңҚеҠЎеҷЁгҖҒеӨҮд»ҪжңҚеҠЎеҷЁ
  3. иҜҒд№ҰжңҚеҠЎгҖҒиҮӘеҠЁеҢ–еҸ‘еёғиҠӮзӮ№гҖҒзӣ‘жҺ§иҠӮзӮ№
  4. еҹҹжҺ§

еҺҹеӣ жҳҜпјҡ

  • иҝҷдәӣжңәеҷЁжӣҙе®№жҳ“д»ҺвҖңжңҚеҠЎзҘЁжҚ®вҖқиҝҮжёЎеҲ°вҖңй«ҳд»·еҖјеҮӯжҚ®вҖқ
  • дёҖж—ҰиҗҪең°пјҢиғҪдёәдёӢдёҖи·іжҸҗдҫӣжӣҙеӨҡжң¬ең°з®ЎзҗҶе‘ҳеҮӯжҚ®гҖҒи®ЎеҲ’д»»еҠЎгҖҒRDP/WinRMгҖҒиҜҒд№ҰжҲ–иҝҗз»ҙд»ӨзүҢ
  • зӣҙжҺҘжү“еҹҹжҺ§зҡ„ж—Ҙеҝ—еҺӢеҠӣе’Ңе‘ҠиӯҰжҰӮзҺҮжӣҙй«ҳ

еӣ жӯӨ RBCD зҡ„жңҖдҪіеҲ©з”Ёе§ҝеҠҝеҫҖеҫҖдёҚжҳҜвҖңз«ӢеҲ»жү“ DCвҖқпјҢиҖҢжҳҜз”Ёе®ғеҒҡдёҖи·ізІҫзЎ® takeoverгҖӮ

2. еҲ©з”Ёй“ҫдёҖпјҡACL еҶҷе…Ҙ RBCD

иҝҷжҳҜжё—йҖҸжөӢиҜ•йҮҢжңҖе…ёеһӢгҖҒжңҖзЁіе®ҡзҡ„дёҖжқЎй“ҫгҖӮе…¬ејҖиө„ж–ҷгҖҒе®һжҲҳеҚҡе®ўе’Ңеёёи§Ғе·Ҙе…·йғҪеӣҙз»•иҝҷжқЎи·Ҝеҫ„еұ•ејҖгҖӮ

2.1 еңәжҷҜи®ҫе®ҡ

еҒҮи®ҫеҪ“еүҚе·ІиҺ·еҫ—жҷ®йҖҡеҹҹз”ЁжҲ· lowprivпјҢе№¶еҸ‘зҺ°пјҡ

  • еҹҹй»ҳи®Өе…Ғи®ёеҲӣе»әжңәеҷЁиҙҰжҲ·
  • lowpriv еҜ№ FILE01$ и®Ўз®—жңәеҜ№иұЎе…·еӨҮеҶҷжқғйҷҗ

жӯӨж—¶е°ұеҸҜд»ҘзӣҙжҺҘиө°ж ҮеҮҶ RBCD й“ҫгҖӮ

иҝҷдёӘеңәжҷҜд№ҹжҳҜ harmj0y и®Ўз®—жңәжҺҘз®ЎжЎҲдҫӢзҡ„ж ёеҝғжҖқи·ҜпјҡеҰӮжһңдёҖдёӘдҪҺжқғйҷҗдё»дҪ“еҜ№жҹҗи®Ўз®—жңәеҜ№иұЎе…·еӨҮи¶іеӨҹзҡ„еҜ№иұЎеҶҷжқғйҷҗпјҢйӮЈе°ұж„Ҹе‘ізқҖиҝҷеҸ°жңәеҷЁжң¬иә«е·Із»ҸеҸҜд»Ҙиў«жҺҘз®ЎгҖӮ

2.1.1 ејҖе§ӢеүҚе»әи®®еҒҡзҡ„ж ЎйӘҢ

зңҹжӯЈеҠЁжүӢеүҚжңҖеҘҪеӨҡеҒҡдёӨжӯҘпјҢдёҚз„¶еҫҲе®№жҳ“жҠҠзҺҜеўғзҠ¶жҖҒжҗһд№ұпјҡ

# иҜ»еҸ–зӣ®ж ҮеҪ“еүҚ RBCD й…ҚзҪ®
rbcd.py -delegate-to 'FILE01$' -action read -dc-ip 10.10.10.10 \
  'corp.local/lowpriv:Password123!'

д»ҘеҸҠпјҡ

Get-ADUser Administrator -Properties AccountNotDelegated,MemberOf

еҰӮжһңдҪ еҮҶеӨҮз”ЁжҹҗдёӘйқһеҶ…зҪ®з®ЎзҗҶе‘ҳеҒҡ impersonationпјҢиҰҒе…ҲзЎ®и®Өд»–жІЎжңүиў«ж Үи®°дёәвҖңж•Ҹж„ҹдё”дёҚиғҪиў«е§”жҙҫвҖқгҖӮ

2.1.2 иҜ·жұӮдёҺе“Қеә”и®°еҪ•ж–№ејҸ

дёәдәҶи®©иҝҷзұ»ж–Үз« зңҹжӯЈе…·еӨҮеӨҚзҺ°д»·еҖјпјҢе»әи®®еңЁе®һжҲҳи®°еҪ•йҮҢжҠҠжҜҸдёҖжӯҘйғҪжӢҶжҲҗдёӨйғЁеҲҶпјҡ

  • иҜ·жұӮпјҡдҪ еҸ‘иө·дәҶд»Җд№Ҳе‘Ҫд»ӨгҖҒзӣ®ж ҮжҳҜи°ҒгҖҒе…ій”®еҸӮж•°жҳҜд»Җд№Ҳ
  • е“Қеә”пјҡе·Ҙе…·иҝ”еӣһдәҶд»Җд№ҲгҖҒе“Әдәӣеӯ—ж®өиҜҙжҳҺжҲҗеҠҹгҖҒе“Әдәӣеӯ—ж®өиҜҙжҳҺеӨұиҙҘ

еңЁ RBCD еңәжҷҜдёӢпјҢиҮіе°‘еә”и®°еҪ•дёӢйқўеҮ з»„иҜ·жұӮдёҺе“Қеә”пјҡ

  1. ж–°е»әжңәеҷЁиҙҰжҲ·
  2. иҜ»еҸ–жҲ–еҶҷе…Ҙ msDS-AllowedToActOnBehalfOfOtherIdentity
  3. з”іиҜ· S4U2Self + S4U2Proxy зҘЁжҚ®
  4. дҪҝз”ЁзҘЁжҚ®и®ҝй—®зӣ®ж ҮжңҚеҠЎ
  5. дәӢеҗҺеңЁж—Ҙеҝ—дҫ§еҜ№еә”еҲ°е“ӘжқЎ 5136 / 4741 / 4769

иҝҷж ·ж—ўиғҪи®©зәўйҳҹеӨҚзҺ°е®һйӘҢпјҢд№ҹиғҪи®©и“қйҳҹеҒҡж”»еҮ»й“ҫеӣһж”ҫгҖӮ

2.2 з¬¬дёҖжӯҘпјҡеҲӣе»әж”»еҮ»иҖ…жҺ§еҲ¶зҡ„жңәеҷЁиҙҰжҲ·

  • жңәеҷЁиҙҰжҲ·еҗҚ
  • жңәеҷЁиҙҰжҲ·еҜҶз Ғ
  • еҜ№еә” SID

еҗҺз»ӯеҶҷе…ҘеұһжҖ§жң¬иҙЁдёҠжҳҜжҠҠиҝҷдёӘжңәеҷЁдё»дҪ“зҡ„ SID еҶҷеҲ°зӣ®ж ҮеҜ№иұЎзҡ„е®үе…ЁжҸҸиҝ°з¬ҰйҮҢгҖӮ

4. жӣҙй«ҳзә§дҪҶжӣҙе°‘и§Ғзҡ„еҸҳдҪ“

4.1 SPN-less RBCD

иҝ‘е№ҙзҡ„е…¬ејҖз ”з©¶е·Із»ҸиҜҒжҳҺпјҢRBCD е№¶дёҚж°ёиҝңејәдҫқиө–вҖңж”»еҮ»иҖ…еҝ…йЎ»жҺ§еҲ¶дёҖдёӘеёҰ SPN зҡ„жңҚеҠЎиҙҰжҲ·вҖқгҖӮжҹҗдәӣеңәжҷҜдёӢеҸҜд»ҘеҲ©з”Ё U2U зӣёе…іжҠҖе·§е®һзҺ° SPN-less RBCDпјҢеҚідҪҝ MachineAccountQuota=0 д№ҹеҸҜиғҪжүҫеҲ°з»•иЎҢи·Ҝеҫ„гҖӮ

дҪҶиҝҷзұ»ж–№жі•е®һжҲҳжҲҗжң¬жӣҙй«ҳпјҢзҺҜеўғзәҰжқҹжӣҙеӨҡпјҢиҖҢдё”еҸҜиғҪеҪұе“Қиў«еҲ©з”ЁиҙҰжҲ·жӯЈеёёдҪҝз”ЁгҖӮеҜ№дәҺеёёи§„жё—йҖҸжөӢиҜ•пјҢдјҳе…Ҳзә§д»Қз„¶дҪҺдәҺвҖңж–°е»әжңәеҷЁиҙҰжҲ· + ж ҮеҮҶ S4UвҖқгҖӮ

жҚўеҸҘиҜқиҜҙпјҢSPN-less RBCD жӣҙеғҸжҳҜвҖңеҪ“ж ҮеҮҶи·Ҝзәҝиў« MachineAccountQuota=0 еҚЎжӯ»ж—¶зҡ„еӨҮз”Ёж–№жЎҲвҖқпјҢиҖҢдёҚжҳҜеӨ§еӨҡж•°йЎ№зӣ®йҮҢзҡ„дё»жөҒи·ҜзәҝгҖӮ

4.2 и·ЁеҹҹдёҺи·Ёжһ—еңәжҷҜ

йғЁеҲҶе…¬ејҖиө„ж–ҷе·Із»Ҹи®Ёи®әи·Ёеҹҹ RBCD зҡ„з»ҶиҠӮй—®йўҳпјҢдҫӢеҰӮпјҡ

  • Linux е·Ҙе…·й“ҫй»ҳи®ӨжөҒзЁӢдёҚдёҖе®ҡиғҪжӯЈзЎ®еӨ„зҗҶи·Ёеҹҹ S4U
  • иў«жЁЎжӢҹз”ЁжҲ·йҖҡеёёиҰҒдёҺж”»еҮ»иҖ…жҺ§еҲ¶дё»дҪ“еӨ„дәҺе…је®№зҡ„дҝЎд»»иҢғеӣҙеҶ…

еӣ жӯӨи·Ёеҹҹ RBCD дёҚжҳҜвҖңе·Ҙе…·дёҖжҠҠжўӯвҖқпјҢиҖҢжҳҜйңҖиҰҒе…ҲзЎ®и®ӨдҝЎд»»е…ізі»гҖҒеҹҹиҫ№з•Ңе’Ң KDC иЎҢдёәгҖӮ

еҰӮжһңж”»еҮ»иҖ…иө°зҡ„жҳҜй»ҳи®Ө MachineAccountQuota и·ҜзәҝпјҢйҖҡеёёдјҡеңЁеҹҹжҺ§дёҠз•ҷдёӢ 4741гҖӮ

иҝҷдёҖдәӢд»¶еҚ•зӢ¬зңӢе№¶дёҚдёҖе®ҡжҒ¶ж„ҸпјҢдҪҶз»“еҗҲдёҠдёӢж–Үд»·еҖјжһҒй«ҳпјҡ

  • еҲӣе»әиҖ…жҳҜжҷ®йҖҡз”ЁжҲ·иҖҢдёҚжҳҜеҠ еҹҹжөҒзЁӢиҙҰеҸ·

  • еҶҚзңӢиҜҘжңәеҷЁиҙҰжҲ·жҳҜеҗҰеҫҲеҝ«еҮәзҺ°еңЁ 4769

5.3 Event ID 4769пјҡS4U2Self дёҺ S4U2Proxy

иҝҷжҳҜ RBCD жңҖе…·иЎҢдёәзү№еҫҒзҡ„йғЁеҲҶгҖӮ

S4U2Self зҡ„еёёи§ҒжЈҖжөӢжҖқи·Ҝпјҡ

  • Account Name дёҺ Service Name жҢҮеҗ‘зӣёеҗҢдё»дҪ“
  • зҘЁжҚ®иҜ·жұӮдё»дҪ“еҫҖеҫҖжҳҜжңәеҷЁиҙҰжҲ·жҲ–еёҰ SPN зҡ„жңҚеҠЎиҙҰжҲ·

S4U2Proxy зҡ„еёёи§ҒжЈҖжөӢжҖқи·Ҝпјҡ

  • Transited Services йқһз©ә
  • зҙ§жҺҘзқҖеҮәзҺ°йқўеҗ‘зӣ®ж ҮжңҚеҠЎзҡ„ TGS иҜ·жұӮ
  • иҜ·жұӮдё»дҪ“дёҚжҳҜж—Ҙеёёе§”жҙҫи®ҫеӨҮпјҢеҚҙеңЁиҜ·жұӮй«ҳд»·еҖјзӣ®ж ҮжңҚеҠЎ

е…¬ејҖжЈҖжөӢж–Үз« жҷ®йҒҚе»әи®®жҠҠдёӢйқўеӣӣзұ»еҠЁдҪңе…іиҒ”иө·жқҘзңӢпјҡ

  1. 4741 ж–°жңәеҷЁиҙҰжҲ·еҲӣе»ә
  2. 5136 дҝ®ж”№ msDS-AllowedToActOnBehalfOfOtherIdentity
  3. з¬¬дёҖжқЎ 4769пјҡS4U2Self
  4. з¬¬дәҢжқЎ 4769пјҡS4U2Proxy

еҰӮжһңеӣӣжӯҘеңЁиҫғзҹӯж—¶й—ҙзӘ—еҸЈеҶ…дёІиө·жқҘпјҢеҹәжң¬е°ұжҳҜй«ҳзҪ®дҝЎеәҰ RBCD еҲ©з”Ёй“ҫгҖӮ

swolfsec зҡ„е…¬ејҖеҲҶжһҗиҝҳзү№еҲ«жҸҗеҲ°дәҶдёӨдёӘеӯ—ж®өеұӮзү№еҫҒпјҡ

  • S4U2Self еңәжҷҜйҮҢпјҢAccount Name е’Ң Service Name дјҡжҢҮеҗ‘еҗҢдёҖдё»дҪ“
  • S4U2Proxy еңәжҷҜйҮҢпјҢTransited Services еҫҖеҫҖйқһз©ә

жӯӨеӨ–пјҢS4U2Self еёёи§Ғ TicketOptions еҸҜи§Ғ 0x40800018 иҝҷзұ»еҖјгҖӮеҚ•зӢ¬жӢҝе®ғеҒҡејәжЈҖжөӢжңӘеҝ…зЁіпјҢдҪҶдҪңдёәи°ғжҹҘиҫ…еҠ©еӯ—ж®өеҫҲжңүд»·еҖјгҖӮ

5.4 зӣ®ж Үдё»жңәдҫ§зҡ„дәҢж¬Ўз—•иҝ№

RBCD жң¬иә«еҸӘиҙҹиҙЈжӢҝзҘЁжҚ®пјҢдёҚиҙҹиҙЈжңҖз»Ҳе…ҘдҫөеҠЁдҪңгҖӮзңҹжӯЈзҡ„еҗҺз»ӯеҚұе®іеҫҖеҫҖдҪ“зҺ°еңЁзӣ®ж Үдё»жңәдёҠпјҡ

  • 4624 зҪ‘з»ңзҷ»еҪ•
  • 5140 е…ұдә«и®ҝй—®
  • 7045 ж–°жңҚеҠЎе®үиЈ…
  • 4698 и®ЎеҲ’д»»еҠЎеҲӣе»ә
  • Sysmon Event ID 1 еҸҜз–‘иҝӣзЁӢеҗҜеҠЁ

жүҖд»ҘдёҖж—ҰеҹҹжҺ§дҫ§еҸ‘зҺ°еҸҜз–‘ 4769пјҢи“қйҳҹеҝ…йЎ»з«ӢеҲ»жҠҠжҺ’жҹҘжү©еұ•еҲ°зӣ®ж Үдё»жңәпјҡ

  • жҳҜеҗҰзҙ§жҺҘзқҖеҮәзҺ°жқҘиҮӘејӮеёёжәҗдё»жңәзҡ„ Kerberos зҷ»еҪ•
  • жҳҜеҗҰжңү SMBгҖҒPsExecгҖҒWinRMгҖҒWMI и®ҝй—®иЎҢдёә
  • жҳҜеҗҰеҮәзҺ°еҮӯжҚ®иҪ¬еӮЁгҖҒиҝңзЁӢжңҚеҠЎеҲӣе»әжҲ–д»»еҠЎжҠ•йҖ’

зӣ®ж Үдё»жңәе“Қеә”и®°еҪ•зӨәдҫӢ

5.6 еҗҲжі•дҪҝз”ЁдёҺиҜҜжҠҘиҫ№з•Ң

дёҚиғҪз®ҖеҚ•жҠҠжүҖжңү RBCD зӣёе…ідәӢд»¶йғҪзӯүеҗҢжҒ¶ж„ҸгҖӮеңЁжҹҗдәӣе°‘ж•°дёҡеҠЎзҺҜеўғйҮҢпјҢе§”жҙҫзЎ®е®һиў«еҗҲжі•дҪҝз”ЁгҖӮдҪҶиҝҷзұ»еҗҲжі•еңәжҷҜйҖҡеёёе…·еӨҮеҮ дёӘзЁіе®ҡзү№еҫҒпјҡ

  • еӣәе®ҡзҡ„е°‘ж•°жңҚеҠЎеҷЁеҸӮдёҺ
  • еӣәе®ҡзҡ„жңҚеҠЎиҙҰжҲ·еҸӮдёҺ
  • еұһжҖ§дҝ®ж”№жһҒе°‘еҸ‘з”ҹ
  • е§”жҙҫй“ҫй•ҝжңҹдёҚеҸҳ

еӣ жӯӨеҜ№и“қйҳҹжқҘиҜҙпјҢжңҖйҮҚиҰҒзҡ„дёҚжҳҜвҖңзҹҘйҒ“иҝҷдёӘеҗҚиҜҚвҖқпјҢиҖҢжҳҜе»әз«Ӣеҹәзәҝпјҡ

  • е“ӘдәӣеҜ№иұЎе№іж—¶жңүеҗҲжі• RBCD
  • е“ӘдәӣиҙҰжҲ·е№іж—¶дјҡж”№иҝҷдёӘеұһжҖ§
  • е“ӘдәӣжңәеҷЁе№іж—¶дјҡеҮәзҺ° S4U2Self/S4U2Proxy

еҒҸзҰ»еҹәзәҝзҡ„дәӢд»¶пјҢжүҚеә”иҝӣе…Ҙй«ҳдјҳе…Ҳзә§еӨ„зҪ®гҖӮ

6. и°ғжҹҘжөҒзЁӢпјҡжҢүж—¶й—ҙзәҝиҝҳеҺҹ

еҸ‘зҺ°з–‘дјј RBCD еҗҺпјҢдёҚиҰҒеҸӘеҲ еұһжҖ§гҖӮжӯЈзЎ®еҒҡжі•жҳҜжҢүй“ҫи·Ҝеӣһж”ҫгҖӮ

6.1 з¬¬дёҖйҳ¶ж®өпјҡзЎ®и®Өи°Ғж”№дәҶи°Ғ

е…Ҳеӣҙз»• 5136 жӢҝеҲ°иҝҷеҮ дёӘе…ій”®еӯ—ж®өпјҡ

  • Subject Account
  • Object DN
  • иў«дҝ®ж”№еұһжҖ§еҗҚ
  • ж—¶й—ҙжҲі

иҝҷдёҖжӯҘзҡ„зӣ®ж ҮжҳҜзЎ®и®Өпјҡ

  • и°Ғж”№дәҶ RBCD
  • ж”№еҲ°дәҶе“ӘеҸ°жңәеҷЁдёҠ

и°ғжҹҘж—¶е»әи®®еҗҢж—¶жҠҠиҝҷдёӨд»¶дәӢдёҖе№¶й—®жё…пјҡ

  • иҝҷжҳҜйҰ–ж¬Ўдҝ®ж”№пјҢиҝҳжҳҜеҺҶеҸІдёҠе·Іжңүе§”жҙҫе…ізі»
  • дҝ®ж”№иҖ…жҳҜеҗҰжң¬жқҘе°ұеә”еҪ“жҺҘи§ҰиҜҘй«ҳд»·еҖјеҜ№иұЎ

6.2 з¬¬дәҢйҳ¶ж®өпјҡзЎ®и®Өж”»еҮ»иҖ…жҺ§еҲ¶дәҶе“ӘдёӘдё»дҪ“

еҶҚжЈҖжҹҘжҳҜеҗҰеӯҳеңЁпјҡ

  • ж–°жңәеҷЁиҙҰжҲ·еҲӣе»ә 4741
  • зӣёе…іжңәеҷЁиҙҰжҲ·зҡ„еҗҺз»ӯ 4769
  • жңәеҷЁиҙҰжҲ·жҲ–жңҚеҠЎиҙҰжҲ·ејӮеёёжҙ»и·ғ

иҝҷйҮҢеёёиғҪе®ҡдҪҚеҮәж”»еҮ»иҖ…жҺ§еҲ¶зҡ„дёӯй—ҙдё»дҪ“пјҢдҫӢеҰӮ RBCD-SVC$ жҲ–йҡҸжңәж–°е»әжңәеҷЁеҗҚгҖӮ

еҰӮжһңе®ҡдҪҚеҲ°дәҶж–°жңәеҷЁиҙҰжҲ·пјҢдёҚиҰҒеҸӘеҒңз•ҷеңЁвҖңеҲ жҺүе®ғвҖқгҖӮиҝҳиҰҒеӣһзңӢпјҡ

6.3 з¬¬дёүйҳ¶ж®өпјҡзЎ®и®ӨжңҖз»Ҳи®ҝй—®иҗҪеҲ°е“ӘйҮҢ

  • зңӢ 5140гҖҒ7045гҖҒ4698
  • зңӢжҳҜеҗҰжңү secretsdumpгҖҒpsexecгҖҒwmiexec дёҖзұ»еҠЁдҪңзҡ„иҗҪең°з—•иҝ№

6.4 з¬¬еӣӣйҳ¶ж®өпјҡеҗ‘еүҚеӣһжәҜе…ҘеҸЈ

еҰӮжһңжҖҖз–‘жҳҜ Relay -> RBCD и·ҜзәҝпјҢеӨ„зҪ®дёҚиғҪеҸӘеҒңз•ҷеңЁ AD дҫ§гҖӮиҝҳиҰҒеҫҖеүҚеҖ’жҹҘпјҡ

  • жҳҜеҗҰеӯҳеңЁи®ӨиҜҒиҜұеҜјиЎҢдёә

7. еӨ„зҪ®дёҺеҠ еӣә

7.1 з«ӢеҚіеӨ„зҪ®

  • жЈҖжҹҘзӣ®ж Үдё»жңәжҳҜеҗҰе·Іиў«жӢҝдёӢе№¶еҒҡеә”жҖҘйҡ”зҰ»

еҰӮжһңзӣ®ж ҮжҳҜеҹҹжҺ§жҲ–з®ЎзҗҶиҠӮзӮ№пјҢиҝҳеә”иҝӣдёҖжӯҘиҖғиҷ‘пјҡ

  • йҮҚзҪ®иў«жЁЎжӢҹй«ҳжқғйҷҗиҙҰжҲ·еҮӯжҚ®
  • жЈҖжҹҘжҳҜеҗҰе·ІеҜјеҮәзӣ®еҪ•ж•°жҚ®жҲ–жң¬ең°еҮӯжҚ®
  • иҜ„дј°жҳҜеҗҰе·ІжңүдәҢж¬ЎжҢҒд№…еҢ–пјҢжҜ”еҰӮи®ЎеҲ’д»»еҠЎгҖҒжңҚеҠЎгҖҒзҷ»еҪ•и„ҡжң¬гҖҒиҜҒд№Ұж»Ҙз”ЁжҲ–ж–°еҜ№иұЎ ACL

7.2 й•ҝжңҹеҠ еӣә

  • жё…зҗҶжҷ®йҖҡз”ЁжҲ·жҲ–дёҡеҠЎз»„еҜ№жңҚеҠЎеҷЁеҜ№иұЎзҡ„еҶҷжқғйҷҗ
  • з»ҷй«ҳд»·еҖјиҙҰжҲ·и®ҫзҪ®вҖңж•Ҹж„ҹдё”дёҚиғҪиў«е§”жҙҫвҖқ
  • ејәеҲ¶ LDAP Signing

  • еҜ№ 4769 дёӯ S4U2Self + S4U2Proxy з»„еҗҲе»әз«Ӣе…іиҒ”и§„еҲҷ

еҰӮжһңзҹӯжңҹж— жі•еҪ»еә•е®ҢжҲҗжүҖжңүеҸҚдёӯз»§ж”№йҖ пјҢиҮіе°‘д№ҹиҰҒдјҳе…ҲиҗҪе®һдёӨд»¶дәӢпјҡ

  • е°Ҷ MachineAccountQuota ж”№дёә 0
  • еҜ№ msDS-AllowedToActOnBehalfOfOtherIdentity зҡ„дҝ®ж”№е»әз«Ӣдё“й—Ёй«ҳдјҳе…Ҳзә§е®Ўи®Ў

еҮ дёӘеёёи§ҒиҜҜеҢәпјҡ

  • еҸӘжҠҠ 5136 еҪ“жҲҗвҖңAD иҝҗз»ҙж“ҚдҪңвҖқ
  • еҸӘзңӢ 4624пјҢеҝҪз•Ҙ 4769
  • зңӢеҲ°ж–°жңәеҷЁиҙҰжҲ·еҲӣе»әдҪҶдёҚе…іиҒ”еҗҺз»ӯзҘЁжҚ®иЎҢдёә
  • еҲ йҷӨ RBCD еұһжҖ§еҗҺе°ұз»“жқҹи°ғжҹҘпјҢеҝҪз•Ҙзӣ®ж Үдё»жңәеҸҜиғҪе·Іиў«жҺҘз®Ў

8. еӨҚзӣҳжё…еҚ•

еҰӮжһңжҠҠиҝҷзҜҮж–Үз« иҗҪеҲ°зңҹе®һйЎ№зӣ®дёӯпјҢжңҖжңүд»·еҖјзҡ„дёҚжҳҜжҰӮеҝөпјҢиҖҢжҳҜеӨҚзӣҳж—¶жңүжІЎжңүдёҖеј иғҪзӣҙжҺҘеҜ№з…§зҡ„жё…еҚ•гҖӮ

8.1 зәўйҳҹдҫ§жңҖе°ҸеҲӨж–ӯйӣҶ
