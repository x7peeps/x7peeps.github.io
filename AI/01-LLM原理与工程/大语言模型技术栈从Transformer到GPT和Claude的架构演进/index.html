еӨ§иҜӯиЁҖжЁЎеһӢжҠҖжңҜж Ҳпјҡд»Һ Transformer еҲ° GPT/Claude зҡ„жһ¶жһ„жј”иҝӣ

еӨ§иҜӯиЁҖжЁЎеһӢжҠҖжңҜж Ҳпјҡд»Һ Transformer еҲ° GPT/Claude зҡ„жһ¶жһ„жј”иҝӣ

еӨ§иҜӯиЁҖжЁЎеһӢпјҲLLMпјүе·Ід»ҺеӯҰжңҜи®әж–Үдёӯзҡ„е®һйӘҢжҖ§е·ҘдҪңпјҢжј”еҸҳдёәж”Ҝж’‘ж•°еҚҒдәҝз”ЁжҲ·ж—ҘеёёдәӨдә’зҡ„еҹәзЎҖи®ҫж–Ҫзә§дә§е“ҒгҖӮдҪңдёә AI еә”з”ЁејҖеҸ‘иҖ…пјҢзҗҶи§Јд»Һеә•еұӮ Transformer жһ¶жһ„еҲ°дёҠеұӮеә”з”ЁжЎҶжһ¶зҡ„е®Ңж•ҙжҠҖжңҜж ҲпјҢжҳҜеҒҡеҮәжӯЈзЎ®жҠҖжңҜйҖүеһӢгҖҒжһ„е»әй«ҳиҙЁйҮҸ AI еә”з”Ёзҡ„еүҚжҸҗгҖӮ

жң¬ж–ҮдёҚи®Ёи®ә"д»Җд№ҲжҳҜ AI"иҝҷзұ»е…Ҙй—ЁжҰӮеҝөпјҢиҖҢжҳҜд»ҘжҠҖжңҜж Ҳзәөи§Ҳзҡ„ж–№ејҸпјҢзі»з»ҹжўізҗҶ LLM з”ҹжҖҒдёӯзҡ„е…ій”®жҠҖжңҜиҠӮзӮ№пјҡд»Һ Attention жңәеҲ¶зҡ„жң¬иҙЁпјҢеҲ°еҗ„дё»жөҒжЁЎеһӢе®¶ж—Ҹзҡ„жһ¶жһ„жј”иҝӣпјҢеҶҚеҲ°е·ҘзЁӢиҗҪең°ж—¶еҝ…йЎ»йқўеҜ№зҡ„ Token еҢ–гҖҒдёҠдёӢж–ҮзӘ—еҸЈгҖҒжҺЁзҗҶйғЁзҪІзӯүж ёеҝғй—®йўҳгҖӮ

1. Transformer жһ¶жһ„ж ёеҝғ

2017 е№ҙ Google еҸ‘иЎЁзҡ„ Attention Is All You Need жҳҜеҪ“еүҚ LLM ж—¶д»Јзҡ„иө·зӮ№гҖӮTransformer д№ӢжүҖд»ҘиғҪжӣҝд»Ј RNN/LSTM жҲҗдёәдё»жөҒеәҸеҲ—е»әжЁЎжһ¶жһ„пјҢж ёеҝғеңЁдәҺдёүдёӘи®ҫи®ЎеҶізӯ–пјҡ

1.1 Self-Attention зҡ„жң¬иҙЁзӣҙи§ү

Self-Attention зҡ„ж ёеҝғж“ҚдҪңеҸҜд»Ҙз”ЁдёҖеҸҘиҜқжҰӮжӢ¬пјҡеҜ№еәҸеҲ—дёӯзҡ„жҜҸдёӘ TokenпјҢеҠЁжҖҒи®Ўз®—е®ғдёҺжүҖжңүе…¶д»– Token зҡ„зӣёе…іжҖ§жқғйҮҚпјҢз„¶еҗҺжҢүжқғйҮҚеҠ жқғиҒҡеҗҲдҝЎжҒҜгҖӮ

дёҺ RNN зҡ„йҖҗжӯҘдј йҖ’дёҚеҗҢпјҢSelf-Attention жҳҜе®Ңе…Ёе№¶иЎҢзҡ„вҖ”вҖ”еәҸеҲ—дёӯзҡ„жҜҸдёӘдҪҚзҪ®еҗҢж—¶е…іжіЁжүҖжңүе…¶д»–дҪҚзҪ®пјҢиҝҷдҪҝеҫ— GPU зҡ„еӨ§и§„жЁЎе№¶иЎҢи®Ўз®—иғҪеҠӣеҫ—д»Ҙе……еҲҶеҲ©з”ЁгҖӮ

еҜ№дәҺ QueryпјҲQпјүгҖҒKeyпјҲKпјүгҖҒValueпјҲVпјүдёүдёӘеҗ‘йҮҸзҡ„и§’иүІпјҢдёҖдёӘе®һз”Ёзҡ„зӣҙи§үжҳҜпјҡ

  • Q д»ЈиЎЁ"жҲ‘жӯЈеңЁеҜ»жүҫд»Җд№Ҳ"
  • K д»ЈиЎЁ"жҲ‘иғҪжҸҗдҫӣд»Җд№ҲеҢ№й…ҚдҝЎжҒҜ"
  • V д»ЈиЎЁ"еҢ№й…ҚжҲҗеҠҹеҗҺжҲ‘е®һйҷ…жҸҗдҫӣзҡ„еҶ…е®№"

жіЁж„ҸеҠӣеҲҶж•° softmax(QK^T / вҲҡd_k) В· V жң¬иҙЁдёҠжҳҜдёҖдёӘеҸҜеҫ®еҲҶзҡ„иҪҜи·Ҝз”ұжңәеҲ¶пјҡжҜҸдёӘдҪҚзҪ®ж №жҚ® Q-K еҢ№й…ҚеәҰпјҢд»ҺжүҖжңүдҪҚзҪ®зҡ„ V дёӯиҒҡеҗҲдҝЎжҒҜгҖӮ

1.2 Multi-Head Attention зҡ„и®ҫи®ЎеҠЁжңә

еҚ•дёӘ Attention Head еҸӘиғҪжҚ•жҚүдёҖз§Қзұ»еһӢзҡ„е…ізі»жЁЎејҸгҖӮMulti-Head Attention е°Ҷ QгҖҒKгҖҒV жҠ•еҪұеҲ°еӨҡдёӘдҪҺз»ҙеӯҗз©әй—ҙпјҢи®©дёҚеҗҢзҡ„ Head еҗ„иҮӘеӯҰд№ дёҚеҗҢзҡ„иҜӯд№үе…ізі»жЁЎејҸпјҲеҰӮиҜӯжі•дҫқиө–гҖҒжҢҮд»Јж¶Ҳи§ЈгҖҒеұҖйғЁе…ұзҺ°зӯүпјүпјҢжңҖеҗҺжӢјжҺҘиҫ“еҮәгҖӮ

е®һйҷ…е·ҘзЁӢдёӯпјҢеӨ§жЁЎеһӢйҖҡеёёдҪҝз”Ё 32-128 дёӘ Attention HeadпјҢжҜҸдёӘ Head зҡ„з»ҙеәҰеңЁ 64-128 д№Ӣй—ҙгҖӮиҝҷз§Қи®ҫи®ЎеңЁдёҚеўһеҠ еҚ•ж¬Ў Attention и®Ўз®—еӨҚжқӮеәҰзҡ„еүҚжҸҗдёӢпјҢеӨ§е№…еўһејәдәҶжЁЎеһӢзҡ„иЎЁиҫҫиғҪеҠӣгҖӮ

1.3 Feed-Forward NetworkпјҲFFNпјү

Attention еұӮиҙҹиҙЈ Token й—ҙзҡ„дҝЎжҒҜдәӨдә’пјҢFFN еҲҷиҙҹиҙЈеҜ№жҜҸдёӘдҪҚзҪ®зҡ„иЎЁзӨәеҒҡйқһзәҝжҖ§еҸҳжҚўгҖӮзҺ°д»Ј LLM зҡ„ FFN йҖҡеёёйҮҮз”Ё SwiGLU жҲ– GELU жҝҖжҙ»еҮҪж•°пјҢеҸӮж•°йҮҸдёҖиҲ¬дёә Attention еұӮзҡ„ 2-3 еҖҚпјҲд»Ҙ LLaMA дёәдҫӢпјҢFFN зҡ„ hidden_dim = 8/3 Г— d_modelпјүгҖӮ

1.4 дҪҚзҪ®зј–з Ғпјҡз»қеҜ№ vs зӣёеҜ№

Transformer зҡ„ Attention жңәеҲ¶жң¬иә«жҳҜзҪ®жҚўдёҚеҸҳзҡ„вҖ”вҖ”жү“д№ұиҫ“е…Ҙ Token зҡ„йЎәеәҸпјҢиҫ“еҮәдёҚеҸҳгҖӮдҪҚзҪ®зј–з Ғи§ЈеҶізҡ„е°ұжҳҜиҝҷдёӘй—®йўҳгҖӮ

ж–№жЎҲд»ЈиЎЁжЁЎеһӢеҺҹзҗҶзү№зӮ№
з»қеҜ№жӯЈејҰзј–з ҒеҺҹе§Ӣ TransformerгҖҒGPT-2з”ЁдёҚеҗҢйў‘зҺҮзҡ„жӯЈејҰ/дҪҷејҰеҮҪж•°дёәжҜҸдёӘдҪҚзҪ®з”ҹжҲҗеӣәе®ҡеҗ‘йҮҸз®ҖеҚ•пјҢдҪҶжіӣеҢ–еҲ°и®ӯз»ғж—¶жңӘи§ҒиҝҮзҡ„еәҸеҲ—й•ҝеәҰж—¶иЎЁзҺ°е·®
еҸҜеӯҰд№ з»қеҜ№зј–з ҒGPT-3гҖҒBERTдёәжҜҸдёӘдҪҚзҪ®еӯҰд№ дёҖдёӘзӢ¬з«Ӣзҡ„ Embedding еҗ‘йҮҸзҒөжҙ»пјҢдҪҶеҗҢж ·еҸ—йҷҗдәҺеӣәе®ҡжңҖеӨ§й•ҝеәҰ
RoPELLaMAгҖҒQwenгҖҒDeepSeekйҖҡиҝҮж—ӢиҪ¬зҹ©йҳөе°ҶдҪҚзҪ®дҝЎжҒҜзј–з ҒеҲ° Q е’Ң K дёӯпјҢдҪҝеҫ— Attention еҲҶж•°иҮӘз„¶еҸҚжҳ зӣёеҜ№и·қзҰ»ж”ҜжҢҒй•ҝеәҰеӨ–жҺЁпјҢеҪ“еүҚдё»жөҒж–№жЎҲ
ALiBiBLOOMгҖҒMPTзӣҙжҺҘеҜ№ Attention еҲҶж•°еҠ дёҖдёӘдёҺи·қзҰ»жҲҗжҜ”дҫӢзҡ„еҒҸзҪ®йЎ№е®һзҺ°жһҒз®ҖпјҢж— йңҖйўқеӨ–еҸӮж•°пјҢеӨ–жҺЁиғҪеҠӣиҫғеҘҪ
зӣёеҜ№дҪҚзҪ®зј–з ҒT5гҖҒeLECTRAеңЁ Attention и®Ўз®—дёӯжҳҫејҸжіЁе…ҘзӣёеҜ№дҪҚзҪ®еҒҸзҪ®зҗҶи®әдјҳйӣ…пјҢе·ҘзЁӢе®һзҺ°иҫғеӨҚжқӮ

еҪ“еүҚж–°ејҖжәҗжЁЎеһӢеҮ д№Һе…ЁйғЁиҪ¬еҗ‘ RoPEпјҢз»“еҗҲ NTK-aware scaling зӯүжҠҖжңҜе®һзҺ°й•ҝдёҠдёӢж–Үж”ҜжҢҒгҖӮ

2. д»Һ GPT-1 еҲ° GPT-4 зҡ„жј”иҝӣи„үз»ң

OpenAI зҡ„ GPT зі»еҲ—жҳҜ LLM еҸ‘еұ•и„үз»ңдёӯжңҖе…·ж Үеҝ—жҖ§зҡ„жҠҖжңҜи·ҜзәҝпјҢжҜҸдёӘйҳ¶ж®өйғҪи§ЈеҶідәҶдёҖдёӘж ёеҝғй—®йўҳпјҡ

2.1 GPT-1пјҲ2018пјүпјҡиҜҒжҳҺйў„и®ӯз»ғзҡ„еҸҜиЎҢжҖ§

  • ж ёеҝғеҲӣж–°пјҡйҰ–ж¬ЎеӨ§и§„жЁЎйӘҢиҜҒдәҶ"ж— зӣ‘зқЈйў„и®ӯз»ғ + жңүзӣ‘зқЈеҫ®и°ғ"пјҲPre-train + Fine-tuneпјүиҢғејҸзҡ„жңүж•ҲжҖ§гҖӮ
  • жһ¶жһ„пјҡ12 еұӮ Transformer DecoderпјҢ117M еҸӮж•°гҖӮ
  • и®ӯз»ғж•°жҚ®пјҡBookCorpusпјҲзәҰ 7000 жң¬д№ҰпјүгҖӮ
  • е…ій”®з»“и®әпјҡеңЁеӨ§и§„жЁЎиҜӯж–ҷдёҠйў„и®ӯз»ғиҜӯиЁҖжЁЎеһӢпјҢеҸҜд»ҘеӯҰеҲ°йҖҡз”Ёзҡ„иҜӯиЁҖиЎЁзӨәпјҢиҝҒз§»еҲ°дёӢжёё NLP д»»еҠЎж—¶жҳҫи‘—жҸҗеҚҮжҖ§иғҪгҖӮ

2.2 GPT-2пјҲ2019пјүпјҡеҸ‘зҺ° Scaling зҡ„ж¶ҢзҺ°иғҪеҠӣ

  • ж ёеҝғеҲӣж–°пјҡиҜҒжҳҺдәҶеҚ•зәҜеўһеӨ§жЁЎеһӢи§„жЁЎпјҲ1.5B еҸӮж•°пјүе’Ңж•°жҚ®йҮҸпјҢеҸҜд»Ҙи®©жЁЎеһӢеңЁйӣ¶ж ·жң¬пјҲZero-shotпјүжқЎд»¶дёӢе®ҢжҲҗеӨҡз§Қд»»еҠЎпјҢж— йңҖд»»дҪ•еҫ®и°ғгҖӮ
  • и®ӯз»ғж•°жҚ®пјҡWebTextпјҲ40GBпјҢReddit й«ҳиҙЁйҮҸеӨ–й“ҫж–Үжң¬пјүгҖӮ
  • е…ій”®жҠҖжңҜпјҡPre-LayerNormпјҲе°Ҷ LayerNorm з§»еҲ° Attention/FFN д№ӢеүҚпјүпјҢж”№е–„и®ӯз»ғзЁіе®ҡжҖ§гҖӮ
  • дәүи®®пјҡOpenAI жңҖеҲқд»Ҙ"еӨӘеҚұйҷ©"дёәз”ұжӢ’з»қе…¬ејҖе®Ңж•ҙжЁЎеһӢпјҢиҝҷжҲҗдёә AI е®үе…Ёи®Ёи®әзҡ„дёҖдёӘж Үеҝ—жҖ§дәӢд»¶гҖӮ

2.3 GPT-3пјҲ2020пјүпјҡIn-Context Learning зҡ„иҜһз”ҹ

  • ж ёеҝғеҲӣж–°пјҡ175B еҸӮж•°и§„жЁЎдёӢж¶ҢзҺ°еҮә In-Context LearningпјҲдёҠдёӢж–ҮеӯҰд№ пјүиғҪеҠӣвҖ”вҖ”жЁЎеһӢд»…йҖҡиҝҮеңЁ Prompt дёӯз»ҷеҮәеҮ дёӘзӨәдҫӢпјҢе°ұиғҪжү§иЎҢж–°д»»еҠЎпјҢж— йңҖжўҜеәҰжӣҙж–°гҖӮ
  • е…ій”®жҠҖжңҜпјҡдәӨжӣҝдҪҝз”Ё Dense Attention е’Ң Sparse AttentionпјҲSliding Window + GlobalпјүпјҢйҷҚдҪҺй•ҝеәҸеҲ—зҡ„и®Ўз®—жҲҗжң¬гҖӮ
  • ж·ұиҝңеҪұе“ҚпјҡIn-Context Learning зӣҙжҺҘеӮ¬з”ҹдәҶ Prompt Engineering иҝҷдёҖж–°иҢғејҸпјҢж”№еҸҳдәҶдәәдёҺ AI дәӨдә’зҡ„еҹәжң¬ж–№ејҸгҖӮ
  • еұҖйҷҗпјҡжЁЎеһӢиҷҪз„¶иғҪеҠӣејәеӨ§пјҢдҪҶиҫ“еҮәиҙЁйҮҸдёҚеҸҜжҺ§вҖ”вҖ”е®ғеҸӘжҳҜеңЁеҒҡ"дёӢдёҖдёӘ Token йў„жөӢ"пјҢдёҚзҗҶи§Јз”ЁжҲ·зҡ„зңҹе®һж„ҸеӣҫгҖӮ

2.4 InstructGPT / ChatGPTпјҲ2022пјүпјҡRLHF иҢғејҸзҡ„зЎ®з«Ӣ
